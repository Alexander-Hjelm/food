# Dumpling Soup Sauce

> ?

## Tags

- asian, chinese, sauce, condiment, soup

## Ingredients

| Ingredient   | Amount      | Prep     | Optional |
| -------------| ----------- | -------- | -------- |
| Soy sauce    | 2 tbsp      |          |          |
| Vinegar     | 1 tbsp      |          |          |
| Chili oil    | 1 tsp       |          |          |
| Salt         | to taste    |          |          |
| Pepper       | to taste    |          |          |

## Instructions

1. Combine soy sauce, vinegar, and chili oil in a small bowl.  
2. Season with salt and pepper to taste.  
3. Stir well and serve alongside dumpling soup.
