# Shukto

> ?

## Tags

- bengali, indian, vegetarian, main course, curry

## Ingredients

| Ingredient             | Amount       | Prep                        | Optional |
| ---------------------- | ----------- | --------------------------- | -------- |
| Green papaya           | 1 medium    | peeled, deseeded, chopped   |          |
| Bitter gourd           | 3           | chopped                     |          |
| Bottle gourd           | 1 medium    | peeled, sliced              |          |
| Ridge gourd            | 1           | peeled, diced               |          |
| Drumsticks             | 4–5         | cut into 5 cm pieces        | Optional |
| Oil                    | 3–4 tbsp    |                             |          |
| Poppy seeds            | 2 tbsp      | soaked                       |          |
| Mustard seeds          | 1 tsp       |                             | Optional |
| Milk                   | 100 ml      |                             |          |
| Ginger paste           | 1 tbsp      |                             |          |
| Panch phoron           | 1 tsp       | Bengali five-spice mix       |          |
| Bay leaf               | 1–2         |                             |          |
| Dried chili            | 1–2         |                             | Optional |
| Salt                   | to taste    |                             |          |

## Instructions

1. Prepare the papaya: peel, remove seeds, chop into medium pieces, and remove the white inner layer.
2. Chop the bitter gourds, peel and slice the bottle gourd, and peel and dice the ridge gourd.
3. Heat oil over medium heat and sauté the bottle gourd until slightly tender.
4. In a blender, mix soaked poppy seeds, mustard seeds, a little milk, and some water to make a smooth paste.
5. Boil the chopped papaya until just tender.
6. Sauté the bitter gourd in oil until lightly cooked.
7. Add the boiled papaya to the pan with bitter gourd and sauté together.
8. In a separate pan, sauté drumsticks and ridge gourd together until lightly cooked.
9. Heat oil on low heat, add panch phoron, bay leaves, and dried chili, and sauté until aromatic.
10. Add ginger paste to the tempered spices.
11. Return all the vegetables to the pan and sauté them in the spiced oil mixture.
12. Add the poppy seed paste along with a little milk and mix well.
13. Continue cooking until all vegetables are tender and coated with the spices.
14. Adjust salt to taste and serve hot with rice.
