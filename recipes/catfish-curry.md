# Catfish Curry

> Spicy, aromatic, and comforting—a Bengali-style fish curry that makes rice disappear fast.

## Tags

- bengali, indian, seafood, main course, curry

## Ingredients

| Ingredient             | Amount         | Prep                          | Optional |
| ---------------------- | -------------- | ----------------------------- | -------- |
| Catfish                | 500 g          | thawed, coated in turmeric and salt |          |
| Pointed gourds         | 3–4            | tops removed, halved, rinsed  |          |
| Cauliflower            | 1 medium head  | cut into medium florets, rinsed |          |
| Mustard oil             | 3–4 tbsp      |                                |          |
| Turmeric powder         | 2 tsp         |                                |          |
| Brown coriander powder  | 2 tsp         |                                |          |
| Cumin powder            | 2 tsp         |                                |          |
| Chili powder            | 1 tsp         |                                | Optional |
| Pickled ginger          | 1 tsp         |                                | Optional |
| Green chilies           | 3–4           | scored at the ends             | Optional |
| Bay leaf                | 1             |                                |          |
| Cumin seeds             | 1 handful     |                                |          |
| Water                   | 2 cups        | warm                           |          |
| Sugar                   | 2 pinches     |                                | Optional |
| Salt                    | to taste      |                                |          |
| Rice                    | as needed     | cooked                         |          |

## Instructions

1. Thaw the catfish in warm water and coat with turmeric and salt.
2. Prepare the pointed gourds by removing tops, halving, and rinsing.
3. Cut cauliflower into medium florets and rinse.
4. Heat mustard oil in a wok or large pan.
5. Fry the catfish in the hot oil until lightly browned and partially cooked. Remove and set aside.
6. In the same oil, sauté cauliflower until lightly browned.
7. Add the pointed gourds and sauté briefly.
8. Prepare the spice mix: combine turmeric, brown coriander powder, cumin powder, chili powder, and pickled ginger.
9. In the pan, heat cumin seeds and bay leaf on low heat, then add green chilies and the spice mix after 1 minute.
10. Sauté the spices on low heat until the oil separates from the spices.
11. Add a small amount of water from the spice mixture to deglaze and pour it into the pan.
12. Add sugar, cauliflower, and pointed gourds.
13. Pour in 2 cups of warm water and a pinch of salt.
14. Cook the vegetables until tender.
15. Add the catfish back to the pan and simmer until fully cooked.
16. Serve hot with steamed rice.
