# Date Balls

> OH MA GAWD IT'S CHRISTMAS!

## Tags

* dessert, swedish, snack, baking

## Ingredients

| Ingredient      | Amount        | Prep               | Optional |
| --------------- | ------------- | ------------------ | -------- |
| Dates           | 200 g         | de-seeded, chopped |          |
| Almonds         | 80 g          | chopped            |          |
| Pumpkin Seeds   | 40 g          |                    |          |
| Sunflower Seeds | 40 g          | chopped            |          |
| Syrup           | 25 g          |                    |          |
| Cinnamon        | 2–3 g         |                    |          |
| Peanut Butter   | 50 g          |                    |          |
| Coconut Flakes  | 50 g          |                    |          |

## Instructions

1. Combine all ingredients except coconut flakes in a bowl. Mix thoroughly until a sticky, uniform mass forms.
2. Roll the mixture into small balls (about 20–25 g each).
3. Roll the balls in coconut flakes to coat.
4. Refrigerate and serve after 1 hour.
