# Zucchini Balls

> ?

## Tags
- swedish, vegetarian, main course, snack

## Ingredients

| Ingredient        | Amount        | Prep                        | Optional |
| ----------------- | ------------- | ---------------------------- | -------- |
| Zucchini          | 2 medium      | shredded, salted, drained    |          |
| Lime zest         | from 2 limes  |                              |          |
| Breadcrumbs       | 200 ml (2 dl) |                              |          |
| Flour             | 5 tsp         |                              |          |
| Parsley           | handful       | chopped                      |          |
| Garlic cloves     | 4             | pressed                      |          |
| Eggs              | 2             |                              |          |
| Salt              | to taste      |                              |          |
| Black pepper      | to taste      |                              |          |
| Breadcrumbs       | as needed     | for coating                  |          |
| Oil               | as needed     | for frying                   |          |
| Potatoes          | as needed     | baked (for serving)          | Optional |
| Tzatziki          | to taste      | for serving                  | Optional |

## Instructions

1. Shred the zucchinis, mix with salt, and let rest for **30 minutes**. Press out as much water as possible.
2. Prepare the batter by mixing lime zest, breadcrumbs (2 dl), flour, chopped parsley, salt, pepper, pressed garlic, and eggs.
3. Combine the drained zucchini with the batter and mix well.
4. Shape into small balls, coat in breadcrumbs, and fry in oil until golden and crisp.
5. Serve with oven-baked potatoes and tzatziki.
