# Cinnamon Buns

> Hey baybeeeeee. Can I sprinkle my egg yolk on your cinnamon buns? ;)

## Tags

- swedish, dessert, snack, baking, sweet

## Ingredients

| Ingredient        | Amount      | Prep        | Optional |
| ----------------- | ----------- | ----------- | -------- |
| Wheat flour       | 800g        |             |          |
| Yeast             | 25g         |             |          |
| Butter            | 50g         | melted      |          |
| Milk              | 300ml       |             |          |
| Salt              | 1 pinch     |             |          |
| Sugar             | 50ml        |             |          |
| Cardamom          | 1 tsp       |             | Optional |
| Butter            | 50g         | melted      |          |
| Cinnamon          | 2 tsp       |             |          |
| Sugar             | 50ml        |             |          |
| Egg               | 1           | for brushing|          |

## Instructions

1. Mix flour, yeast, melted butter, milk, salt, sugar, and cardamom into a smooth dough. Let it rise until doubled.
2. Melt butter, sugar and cinnamon in a pot until the consistensy is an even paste.
3. Roll out dough, spread the paste evenly, then roll up and cut into buns.
4. Place on baking sheet, brush with egg, and bake at 250Â°C for 14 minutes until golden.
