# Fried Egg Toast

> ?

## Tags

- asian, chinese, breakfast, snack, quick, vegetarian

## Ingredients

| Ingredient       | Amount      | Prep         | Optional |
| ---------------- | ----------- | ------------ | -------- |
| Bread            | 2 slices    |              |          |
| Eggs             | 2           | beaten       |          |
| Sesame seeds     | 1 tsp       |              |          |
| Salt             | To taste    |              |          |
| Black pepper     | To taste    |              |          |
| Lettuce          | A few leaves|              | Optional |
| Tomato           | 1           | sliced       | Optional |
| Other sandwich toppings | As desired |          | Optional |

## Instructions

1. Dip bread slices in beaten eggs, ensuring full coverage.  
2. Heat a pan with a little oil or butter, place the egg-coated bread in the pan.  
3. Sprinkle sesame seeds on top and let cook until golden.  
4. Carefully flip and cook the other side. Season with salt and pepper.  
5. Remove from pan and assemble with remaining sandwich toppings like lettuce, tomato, or any extras.
