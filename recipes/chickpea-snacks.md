# Chickpea Snacks

> ?

## Tags

- snack, vegetarian, vegan, appetizer

## Ingredients

| Ingredient     | Amount      | Prep         | Optional |
| -------------- | ----------- | ------------ | -------- |
| Chickpeas      | 400g        | drained      |          |
| Olive oil      | 2 tbsp      |              |          |
| Salt           | 1 tsp       |              |          |
| Cumin powder   | 1 tsp       |              |          |
| Chili powder   | 1/2 tsp     |              |          |
| Black pepper   | 1/2 tsp     |              |          |

## Instructions

1. Preheat the oven to 200°C.  
2. Toss drained chickpeas with olive oil, salt, cumin powder, chili powder, and black pepper until evenly coated.  
3. Spread chickpeas in a single layer on a baking tray.  
4. Bake for 20–25 minutes, shaking the tray halfway through, until crispy and golden.  
5. Let cool slightly and serve as a crunchy snack.
