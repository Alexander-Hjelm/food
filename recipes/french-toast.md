# French Toast

> You cannot save the batter. Give up.

## Tags

- breakfast, snack, swedish, vegetarian

## Ingredients

| Ingredient      | Amount  | Prep | Optional |
| --------------- | ------- | ---- | -------- |
| Flour           | 0.75 dl |      |          |
| Egg             | 1       |      |          |
| Milk            | 0.75 dl |      |          |
| Bread slices    | 2       |      |          |
| Parmesan cheese | to taste | grated | Optional |
| Cooking fat     | as needed |      |          |

## Instructions

1. Whisk flour, egg, and milk into a smooth batter.
2. Dip bread slices into the batter.
3. Fry in cooking fat over medium-high heat; add parmesan if desired.
