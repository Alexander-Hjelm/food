# Paratha Spice Mix

> ?

## Tags

- indian, spice mix

## Ingredients

| Ingredient          | Amount     | Prep          | Optional |
| ------------------ | ---------- | ------------- | -------- |
| Coriander seeds     | Equal parts |              |          |
| Cumin seeds         | Equal parts |              |          |
| Dried chili         | Equal parts |              |          |
| Bay leaves          | Equal parts |              |          |
| Cinnamon            | Small amount | Optional     |          |
| Cardamom            | Small amount | Optional     |          |
| Fennel seeds        | Equal parts | Whole        |          |

## Instructions

1. Mix all ingredients together in equal parts (except optional cinnamon and cardamom, which can be added lightly).  
2. Roast the mixture in a dry pan over medium heat until aromatic.  
3. Let cool slightly, then crush or grind into a coarse powder.  
4. Store in an airtight container until ready to use for parathas or other Indian dishes.
