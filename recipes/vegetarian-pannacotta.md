# Vegetarian Pannacotta

> Kids these days just don't know how to whip up a good pannacotta. My ass! It sure was a lot damn better back in my day.

## Tags

- italian, dessert

## Ingredients

| Ingredient        | Amount     | Prep     | Optional |
| ----------------- | ---------- | -------- | -------- |
| Full fat cream    | 5 dl       |          |          |
| Vanilla extract   | 1 tbsp     |          |          |
| Sugar             | 0.5 dl     |          |          |
| Dr Oetker Vegegel | 1 bag (4g) |          |          |

## Instructions

1. Mix all ingredients
2. Bring to a boil while stirring
3. Pour into serving bowls and refriedgerate for 2 hours.
4. Top with berries, jam or shredded chocolate
