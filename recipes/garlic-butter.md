# Garlic Butter

> ?

## Tags

- italian, condiment, sauce, snack, appetizer

## Ingredients

| Ingredient    | Amount      | Prep        | Optional |
| ------------- | ----------- | ----------- | -------- |
| Butter        | 100g        |             |          |
| Garlic cloves | 2           | mashed      |          |
| Salt          | to taste    |             |          |
| Parsley       | 1 tbsp      | chopped     |          |
| Chives        | 1 tbsp      | chopped     |          |

## Instructions

1. Mix butter with mashed garlic, salt, parsley, and chives.  
2. Heat lightly in a pan until fragrant and slightly melted.  
3. Serve warm as a spread or topping for bread, vegetables, or grilled dishes.
