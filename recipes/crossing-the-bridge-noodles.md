# Eggplant with Garlic Sauce

> Jesus crossed a bridge made of noodles in a watermelon car and escaped from satan. It's true, I saw it on YouTube!

## Tags

- chinese, asian, main course, soup

## Ingredients

| Ingredient                | Amount      | Prep                          | Optional |
| ------------------------- | ----------- | ----------------------------- | -------- |
| Rice Noodles              | 250 g       |                               |          |
| Chicken stock             | 1.5 l       |                               |          |
| Chinkiang vinegar         | 0.5 tsp     |                               |          |
| Light soy sauce           | 15 ml       |                               |          |
| White pepper              | a pinch     |                               |          |
| Salt                      | a pinch     |
| Aromatic spice / Kucharek | a pinch     |
| Ginger                    | 2 tsp       | Minced                        |          |
| Garlic                    | 2 tbsp ~ 6 cloves | Minced (divided)       |          |
| Soy mince                 | 150 g       |                               |          |
| Chili oil                 | 5–10        | Deseeded, sliced              | Optional |
| Tree ear mushrooms        | 10-20       | Soaked in hot water           | Optional |
| Spring onions             | 2           | Chopped                       | Optional |
| Coriander                 |            | Chopped              | Optional |

## Instructions

1. Make the broth with chicken stock, ginger, soy sauce, vinegar, salt and white pepper. Simmer for 15 minutes. If using mushrooms, add them in for the final 3 minutes.
2. In the meantime, prepare rice noodles according to package directions, then rinse and hold warm.
3. Fry the soy mince with neutral oil, garlic, salt, aromatic spice, salt and white pepper.
4. Once the broth is simmered and ready, place the noodles and meat in the serving bowl, then pour the broth over it.
5. Top with spring onions, coriander and chili oil if desired.
