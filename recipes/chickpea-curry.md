# Chickpea Curry

> Curry so good even your spoon will want a second helping!

## Tags

- indian, vegetarian, main course, curry

## Ingredients

| Ingredient        | Amount      | Prep           | Optional |
| ----------------- | ----------- | -------------- | -------- |
| Ginger            | 1 tbsp      | minced         |          |
| Garlic            | 2 cloves    | minced         |          |
| Yellow onion      | 1           | diced          |          |
| Curry powder      | 2 tsp       |                |          |
| Turmeric          | 1 tsp       |                |          |
| Red chili flakes  | 1/2 tsp     |                |          |
| Cumin             | 1 tsp       |                |          |
| Chickpeas         | 400g        | cooked/canned  |          |
| Coconut milk      | 200 ml      |                |          |
| Crushed tomatoes  | 200 ml      |                |          |
| Vegetable broth   | 1 cube      |                |          |
| Coriander         | 1 tsp       | chopped        |          |
| Salt              | To taste    |                |          |
| Garam masala      | 1 tsp       |                | Optional |
| Spinach leaves    | 50g         |                | Optional |
| Rice              | As needed   | cooked         |          |
| Lemon             | 1 slice     |                | Optional |

## Instructions

1. Heat oil in a pan and saut√© ginger, garlic, and yellow onion until fragrant.  
2. Add curry powder, turmeric, red chili flakes, and cumin; stir for 1-2 minutes.  
3. Add chickpeas, coconut milk, crushed tomatoes, vegetable broth cube, coriander, and salt. Mix well.  
4. Let the curry simmer for 10-15 minutes to allow flavors to meld.  
5. Just before serving, top with garam masala and fresh spinach leaves.  
6. Serve hot with cooked rice and a slice of lemon on the side.
