## Ingredients

| ---------------- | ----------- | -------- | -------- |
| Pizza kit        | 1           |          |          |
| Cheese           | 100–200 g   | grated   |          |
| Avocado          | 1           | sliced   | Optional |
| Arugula          | 30 g        |          | Optional |
| Chili            | 1           | sliced   | Optional |
| Onion            | 1           | sliced   | Optional |

| Chili powder     | to taste    |          | Optional |
| Oregano          | to taste    |          | Optional |
| Curry            | to taste    |          | Optional |

## Instructions

1. Preheat oven according to pizza kit instructions.
2. Spread sauce (from kit) on the base. Add cheese and desired toppings.
3. Sprinkle optional spices. Bake according to kit instructions until crust is golden and cheese melted.
