# White Bean Stew

> No, now it got weird. Now we have to fucking tighten up

## Tags

- vegetarian, main course

## Ingredients

| Ingredient        | Amount      | Prep            | Optional |
| ----------------- | ----------- | --------------- | -------- |
| Canned crushed tomatoes | 400g  |                 |          |
| Canned chickpeas or white beans | 400g | drained   |          |
| Garlic cloves      | 1-2         | crushed or chopped |         |
| Apple              | 1           | peeled and diced |          |
| Chili powder       | 1 tsp       |                 |          |
| Curry powder       | 1 tsp       |                 |          |
| Salt               | to taste    |                 |          |
| Pepper             | to taste    |                 |          |

## Instructions

1. Sauté garlic in a little oil until fragrant.  
2. Add tomatoes, beans, apple, chili, curry, salt, and pepper. Simmer for 10–15 minutes until flavors combine.  
3. Serve hot over cooked rice.
