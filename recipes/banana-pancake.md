# Banana Pancake

> You *cannot* save the batter. Give up.

## Tags

- breakfast, snack, swedish, vegetarian

## Ingredients

| Ingredient        | Amount  | Prep    | Optional |
| ----------------- | ------- | ------- | -------- |
| Flour             | 0.75 dl |         |          |
| Banana            | 1       | mashed  |          |
| Oat milk          | 1 dl    |         |          |
| Cinnamon          | to taste |         |          |
| Cardamom          | to taste |         |          |
| Cooking fat/oil   | as needed |        |          |

## Instructions

1. Whisk flour, mashed banana, oat milk, cinnamon, and cardamom into a smooth batter.
2. Heat cooking fat in a pan over medium-high heat.
3. Pour in batter, cook until edges set, flip, and cook briefly again.
