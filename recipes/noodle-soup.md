# Noodle Soup

> noodle-soup.com. Made by your resident Chinese factory worker.

## Tags

- asian, soup, main course

## Ingredients

| Ingredient     | Amount      | Prep         | Optional |
| -------------- | ----------- | ------------ | -------- |
| Noodles        | 200g        |             |          |
| Mixed vegetables | 200g      | chopped      |          |
| Spring onions  | 2           | sliced       |          |
| Coriander      | handful     | chopped      |          |
| Soy sauce      | 2 tbsp      |             |          |
| Salt           | 1 tsp       |             |          |
| Sesame oil     | 1 tsp       |             |          |
| Pepper         | to taste    |             |          |
| Chili oil      | 1 tsp       |             | Optional |

## Instructions

1. Boil noodles with mixed vegetables and a splash of vegetable broth until cooked.  
2. Season with soy sauce, salt, sesame oil, and pepper.  
3. Serve hot, topped with spring onions, coriander, and a drizzle of chili oil.
