# Kebab Plate

> ?

## Tags

- middle eastern, main course, dinner

## Ingredients

| Ingredient        | Amount | Prep      | Optional |
|------------------ |--------|-----------|----------|
| Iceberg lettuce   | 100g   | shredded  |          |
| Cucumber          | 1      | sliced    |          |
| Tomato            | 1      | diced     |          |
| Red onion         | 1/2    | sliced    | Optional |
| Red cabbage       | 50g    | shredded  | Optional |
| Kebab sauce       | 50ml   |           |          |
| Spicy kebab sauce | 50ml   |           | Optional |
| Yoghurt           | 100ml  |           | Optional |

## Instructions

1. Prepare all vegetables by washing and chopping them as needed: shred the lettuce, slice the cucumber, dice the tomato, and slice the red onion or shred the red cabbage.  
2. Arrange all condiments in small bowls or serving plates.  
3. Serve alongside your cooked kebab meat and let everyone build their own plate with sauces and toppings of choice.
