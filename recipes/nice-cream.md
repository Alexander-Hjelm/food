# Nice cream

> ?

## Tags

- snack, vegetarian, vegan, dessert

## Ingredients

| Ingredient      | Amount   | Prep         | Optional |
| --------------- | -------- | ------------ | -------- |
| Banana          | 100g     |              |          |
| Pineapple       | 100g     |              |          |
| Vanilla extract | 1/2 tsp  |              |          |
| Syrup           | 1 tbsp   |              | Optional |

## Instructions

1. Blend the ingredients together
2. Freeze for 1.5-2 hours, until the consistency is ice-creamy but not frozen solid.
3. Stir and serve
