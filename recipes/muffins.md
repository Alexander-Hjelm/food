# Muffins (12 pieces)

> ?

## Tags

- baking, snack, breakfast, dessert, sweet

## Ingredients

| Ingredient        | Amount        | Prep     | Optional |
| ----------------- | ------------- | -------- | -------- |
| All-purpose flour  | 260g          |          |          |
| Sugar              | 30g           |          |          |
| Baking powder      | 1 tbsp        |          |          |
| Salt               | 1/2 tsp       |          |          |
| Cinnamon           | To taste      |          |          |
| Nutmeg             | To taste      |          |          |
| Vanilla extract    | 1 tsp         |          |          |
| Egg                | 1             | beaten   |          |
| Butter             | 4 tbsp        | melted   |          |
| Milk               | 160g          |          |          |

## Instructions

1. Preheat the oven to 200Â°C.
2. In a bowl, mix all the dry ingredients: flour, sugar, baking powder, salt, cinnamon, and nutmeg.
3. In another bowl, combine the wet ingredients: beaten egg, melted butter, milk, and vanilla extract.
4. Gradually pour the wet mixture into the dry ingredients while stirring gently until just combined. Do not overmix.
5. Spoon the batter into a greased or lined muffin tin, filling each cup about 2/3 full.
6. Bake in the preheated oven for 20 minutes, or until the tops are golden and a toothpick inserted in the center comes out clean.
7. Let cool slightly before serving. Enjoy with coffee or on their own!
