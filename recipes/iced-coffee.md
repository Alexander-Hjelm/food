# Iced Coffee

> I know ya don't smoke weed, I know this. But I'm gonna get you high today 'cause it's Friday, you ain't got no job and you ain't got sh*t to do.

## Tags

- beverage, drink, coffee

## Ingredients

| Ingredient       | Amount       | Prep      | Optional |
| ---------------- | ----------- | -------- | -------- |
| Instant coffee   | 2 tsp       |           |          |
| Sugar            | 1-2 tsp     |           |          |
| Hot water        | 60 ml       |           |          |
| Milk             | 100 ml      |           |          |
| Ice cubes        | As needed   |           |          |

## Instructions

1. Dissolve instant coffee and sugar in hot water.  
2. Add milk and stir well.  
3. Pour over ice and serve immediately.
