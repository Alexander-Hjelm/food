# Boba Tea

> Alexander got tired of getting hit on by adolescent chinese women and offered free boba teas in tea shops, so he figured out how to make it to give himself an exit. Now he is a running boba dispenser. Ask him out and maybe you will get one for yourself.

## Tags

- asian, drink, beverage, snack, sweet

## Ingredients

| Ingredient       | Amount      | Prep          | Optional |
| ---------------- | ----------- | ------------- | -------- |
| Water            | 500ml       |               |          |
| Tapioca pearls   | 50g         |               |          |
| Tea bag          | 1           |               |          |
| Sugar            | 2 tbsp      |               |          |
| Milk             | 250ml       |               |          |

## Instructions

1. Boil half the water in a pot and add tapioca pearls to cover the bottom.  
2. Add the tea bag and simmer until tea is brewed; stir in sugar.  
3. When pearls float, add milk (the remaining half of the water) and serve warm or chilled.
