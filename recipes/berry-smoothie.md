# Berry Smoothie

> I’ll be drinking so much of this shit that I’ll need a nutrition label on my body bag.

## Tags

- drink, smoothie, snack, swedish

## Ingredients

| Ingredient                                       | Amount  | Prep       | Optional |
| ------------------------------------------------ | ------- | ---------- | -------- |
| Mixed smoothie fruit (yellow/red) / Fresh Fruit  | 200 g   | frozen     |          |
| Banana                                           | 1 large | sliced     |          |
| Milk / Coconut milk                              | 0.5 dl  |            |          |
| Vanilla extract                                  | 0.5 tsp |            | Optional |
| Syrup / Sugar                                    | 1 tbsp  |            | Optional |

## Instructions

1. Combine ingredients in a blender.
2. Blend until smooth.
3. Drink slowly. Seriously.
