# Veggie Mini Pancakes

> ?

## Tags

- vegetarian, snack, main course, chinese, asian

## Ingredients

| Ingredient    | Amount      | Prep       | Optional |
| ------------- | ----------- | ---------- | -------- |
| Carrots       | 100g        | shredded   |          |
| Cabbage      | 100g        | thinly sliced |          |
| Eggs          | 2           | beaten     |          |
| Flour         | 50g         |            |          |
| Salt          | To taste    |            |          |
| Pepper        | To taste    |            |          |
| Cumin         | 1 tsp       | ground     |          |
| Oil           | 2 tbsp      | for frying |          |

## Instructions

1. Julienne the carrots and thinly slice the cabbage.  
2. In a bowl, mix the vegetables with eggs, flour, salt, pepper, and cumin until well combined.  
3. Heat oil in a pan over high heat.  
4. Spoon small portions of the batter into the pan and flatten slightly to form mini pancakes.  
5. Fry until golden brown on both sides. Serve hot.
