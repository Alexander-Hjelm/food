# Rice Pancakes

> ?

## Tags

- chinese, asian, breakfast, snack, main course

## Ingredients

| Ingredient       | Amount | Prep    | Optional |
| ---------------- | ------ | ------- | -------- |
| Pancake          | 1      |         |          |
| Egg              | 1      |         |          |
| Garlic sauce     | To taste |       |          |
| Bacon            | To taste | cooked |          |
| Lettuce          | To taste | chopped |          |
| Oil              | As needed |         |          |

## Instructions

1. Heat oil in a pan on high heat and fry the pancake for a few minutes.  
2. Crack an egg over the pancake, cook for 1 minute, then flip.  
3. Top with garlic sauce, bacon, and chopped lettuce. Serve immediately.
