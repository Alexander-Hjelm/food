# Sautéed Sulphur Tuft Mushrooms (Svavelticka)

> ?

## Tags

- vegetarian, appetizer, side dish, swedish

## Ingredients

| Ingredient           | Amount      | Prep           | Optional |
| -------------------- | ----------- | -------------- | -------- |
| Sulphur tuft mushrooms | 300 g     | pre-boiled     |          |
| Oil                   | 2 tbsp    |                |          |
| Garlic cloves         | 2         | minced         |          |
| Red onion             | 1 large   | sliced         |          |
| Salt                  | to taste  |                |          |
| Black pepper          | to taste  |                |          |
| Lemon juice           | 1 tbsp    |                |          |
| Chili flakes          | 1 tsp     |                | Optional |
| Vinegar               | 1 tsp     |                | Optional |
| Spring onion          | 2 stalks  | chopped        | Optional |

## Instructions

1. Pre-boil the sulphur tuft mushrooms briefly to soften them.
2. Heat oil in a pan over high heat and sauté the mushrooms until they are slightly crispy.
3. In a separate pan, lightly brown the minced garlic and sliced red onion.
4. Add the pre-cooked mushrooms to the onions and garlic. Season with salt, black pepper, lemon juice, chili flakes, and a splash of vinegar.
5. Continue cooking together for a few minutes until flavors combine and mushrooms are fully cooked.
6. Stir in the chopped spring onion toward the end of cooking for a fresh, bright finish.
