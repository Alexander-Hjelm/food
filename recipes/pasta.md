# How to: Boil Pasta 

> Gordon Ramsay, throw yourself into a well. This is how real badass rockabilly raggers make pasta. Pussy guaranteed.

## Tags

- italian, guide, pasta, basic

## Ingredients

| Ingredient | Amount | Prep | Optional |
| ---------- | ------ | -----| -------- |
| Pasta      | 100 g  |      |          |
| Water     | 1 L    |      |          |
| Salt       | 1 tbsp |  |          |
| Olive oil   | 1 tbsp |        | Optional |
| Spices (star anice/cloves) | 1 piece |  | Optional |

## Instructions

1. Bring water and salt to a boil; add pasta only when it bubbles.
2. Stir (extra important for spaghetti). Add spices if needed.
3. Cook as directed on the package.
4. Drain the water and mix the pasta with salt, pepper and a little oil.
