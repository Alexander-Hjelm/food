# Pizza Sauce

> Cheesy on my peeny and a sauce-a on my balls.

## Tags

- italian, sauce, main course

## Ingredients

| Ingredient     | Amount      | Prep         | Optional |
| -------------- | ----------- | ------------ | -------- |
| Crushed tomatoes | 400g       |             |          |
| Salt           | 1 tsp       |             |          |
| Pepper         | 1/2 tsp     |             |          |
| Olive oil      | 2 tbsp      |             |          |
| Garlic         | 2 cloves    | minced       |          |
| Basil          | 1 tsp       | chopped      |          |

## Instructions

1. Heat olive oil in a pan and sauté minced garlic until fragrant.  
2. Add crushed tomatoes, salt, pepper, and basil.  
3. Simmer on low heat for 10–15 minutes until thickened.
