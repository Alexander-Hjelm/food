# Saffron Rice (Budget)

> ?

## Tags

- asian, middle eastern, side dish, main course, vegan

## Ingredients

| Ingredient         | Amount        | Prep     | Optional |
| -----------------  | ------------ | -------- | -------- |
| Basmati rice       | 200g          |          |          |
| Salt               | to taste      |          |          |
| Vegetable stock    | 500ml         |          |          |
| Olive oil          | 2 tbsp        |          |          |
| Turmeric           | 1 tsp         |          |          |
| Paprika            | 1 tsp         |          |          |
| Garlic powder      | 1/2 tsp       |          |          |
| Onion powder       | 1/2 tsp       |          |          |

## Instructions

1. Boil the basmati rice in salted vegetable stock until tender and water is absorbed.  
2. In a separate bowl, mix olive oil, turmeric, paprika, garlic powder, and onion powder to make a marinade.  
3. Fold the cooked rice into the marinade until evenly coated. Serve warm.
