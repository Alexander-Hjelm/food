# How to: Pancakes

> Flip it like Beckham

## Tags

- breakfast, dessert, snack, swedish, guide

## Ingredients

| Ingredient       | Amount       | Prep       | Optional |
| ---------------- | ----------- | ---------- | -------- |
| Wheat flour      | 375ml       |           |          |
| Salt             | 3/4 tsp     |           |          |
| Milk             | 900ml       |           |          |
| Eggs             | 4.5         |           |          |
| Butter           | for frying  |           |          |
| Jam, berries, or fruit | as desired |       | Optional |

## Instructions

1. Mix flour, salt, and 1/3 of the milk; then add the remaining milk. Let batter rest 20 min.  
2. Heat a pan on medium (2/3), add a little butter, pour in batter, and cook until the top is set.
3. Carefully loosen edges, wait 10 seconds, flip, cook the other side, and repeat with remaining batter, buttering pan each time.
