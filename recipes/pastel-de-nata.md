# Pastel de Nata

> ?

## Tags

- dessert, pastry, portuguese, sweet, breakfast

## Ingredients

| Ingredient         | Amount       | Prep       | Optional |
| ----------------- | ----------- | ---------- | -------- |
| Puff pastry       | Enough for 18 |                |          |
| Eggs              | 4            | beaten     |          |
| Condensed milk    | 100ml        |            |          |
| Milk              | 150ml        |            |          |
| Sugar             | 45g          |            |          |
| Butter            | for greasing | softened   |          |

## Instructions

1. Preheat the oven to 200°C.  
2. Grease 18 tart molds with butter.  
3. Roll out the puff pastry and line the tart molds.  
4. In a mixing bowl, combine beaten eggs, condensed milk, milk, and sugar until smooth.  
5. Pour the custard mixture into the prepared pastry shells.  
6. Bake for 15–20 minutes, until the custard is set and tops are slightly caramelized.  
7. Allow to cool slightly before serving. Enjoy warm or at room temperature.
