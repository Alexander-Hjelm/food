# Lasagna

> ?

## Tags

- italian, main course, pasta, oven-baked

## Ingredients

| Ingredient            | Amount        | Prep            | Optional |
| --------------------- | ------------- | --------------- | -------- |
| Minced meat           | 1000 g        |                 |          |
| Onions                | 3 large       | chopped         |          |
| Garlic cloves         | 4             | minced          |          |
| Crushed tomatoes      | 800 g         |                 |          |
| Tomato purée          | 200 g         |                 |          |
| Stock cubes           | 2             |                 |          |
| Basil                 | 2 tsp         | dried           |          |
| Thyme                 | 2 tsp         | dried           |          |
| Oregano               | 2 tsp         | dried           |          |
| Sambal oelek          | 100 g         |                 | Optional |
| Salt                  | to taste      |                 |          |
| Black pepper          | to taste      |                 |          |
| Butter                | 120 g         |                 |          |
| Wheat flour           | 150 g         |                 |          |
| Milk                  | 1500 ml       |                 |          |
| Grated cheese         | 150 g         |                 |          |
| Lasagna sheets        | as needed     |                 |          |

## Instructions

1. Preheat the oven to 175°C.
2. Heat a large pan and fry the minced meat until well browned.
3. Add the chopped onions and cook until soft and translucent.
4. Stir in the minced garlic and cook briefly until fragrant.
5. Add the crushed tomatoes, tomato purée, crumbled stock cubes, basil, thyme, oregano, and sambal oelek.
6. Let the sauce simmer gently for about 20–30 minutes. Season with salt and black pepper to taste.
7. Meanwhile, prepare the white sauce by melting the butter in a saucepan.
8. Whisk in the flour and cook for a minute without browning.
9. Gradually whisk in the milk while heating, stirring constantly to avoid lumps.
10. Continue cooking until the sauce thickens, then season with salt and black pepper.
11. Stir in the grated cheese until melted and smooth.
12. Assemble the lasagna in an oven dish by layering meat sauce, white sauce, and lasagna sheets.
13. Finish with a layer of white sauce on top.
14. Bake in the oven for about 40 minutes until golden and bubbling.
15. Let the lasagna rest for 10 minutes before serving so it holds together nicely.
