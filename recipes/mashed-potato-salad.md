# Mashed Potato Salad

> ?

## Tags
- salad, side dish, comfort food

## Ingredients

| Ingredient    | Amount | Prep | Optional |
| ------------- | ------ | ---- | -------- |
| Potatoes      | 4      | Boiled and mashed |          |
| Egg           | 1      | Hard-boiled and chopped |          |
| Peas          | A handful | |          |
| Fresh parsley | A few sprigs | Chopped |          |
| Butter        | 1 tbsp | Melted |          |
| Mayonnaise    | 1 tbsp | |          |
| Salt          | To taste | |          |
| Pepper        | To taste | |          |

## Instructions

1. Boil the potatoes until soft, then mash them.
2. Add the chopped boiled egg, peas, and fresh parsley.
3. Stir in the melted butter and mayonnaise.
4. Season with salt and pepper.
5. Mix until creamy and serve warm or chilled.
