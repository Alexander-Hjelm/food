## Ingredients

| Ingredient      | Amount      | Prep     | Optional |
| --------------- | ----------- | -------- | -------- |
| Yeast           | 25g         |          |          |
| Water           | 600ml       |          |          |
| Flour (White)   | 7dl         |          |          |
| Flour (Graham)  | 7dl         |          |          |
| Salt            | 2 tsp       |          |          |
| Seeds, bran, oats, berries | As desired |  | Optional |

## Instructions

1. Dissolve yeast in warm water.  
2. Mix in flour, salt, and any optional additives to form a dough.  
3. Let dough rise, shape, and bake at 200Â°C for 10 minutes.
