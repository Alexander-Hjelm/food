# Fruit Infusion

> Put a liter of this in front of me and watch me become the happiest kiddo at kindergarten!

## Tags

- tea, beverage, infusion, fruit, sweet

## Ingredients

| Ingredient      | Amount          | Prep                  | Optional |
| --------------- | --------------- | --------------------- | -------- |
| Fruit           | 400g            | cut into small pieces |          |
| Honey           | 1-2 tbsp        |                       |          |
| Water           | 1.5 l           |                       |          |
| Black/green tea | 2 bags or 2 tsp |                       |          |

## Instructions

1. Bring water to a boil
2. Combine all ingredients into a pot
3. Infuse for 10-20 minutes
4. Serve hot or store in fridge for later consumption (chilled)
