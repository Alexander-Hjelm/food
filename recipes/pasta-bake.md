# Pasta Bake

> Best eaten with your hands, like a true Viking!

## Tags

- italian, main course, vegetarian

## Ingredients

| Ingredient        | Amount      | Prep            | Optional |
| ----------------- | ----------- | --------------- | -------- |
| Pasta (fusili)    | 250g        |                 |          |
| Grated cheese     | 300g        |                 |          |
| Garlic cloves     | 4           | pressed         |          |
| Red chili         | 2           | finely chopped  |          |
| Tomatoes          | 2           | diced           |          |
| Oregano           | 2 tsp       |                 | Optional |
| Milk substitute   | 300ml       |                 |          |
| Flour             | 2 tbsp      |                 |          |
| Olive oil         | 2 tbsp      |                 |          |

## Instructions

1. Cook pasta according to package instructions.  
2. Saut√© garlic, chili, and tomatoes in oil over medium heat. Reduce to low, add flour, and stir until thick. Gradually add milk substitute while stirring.  
3. Mix in half the cheese until melted, then repeat with remaining cheese. Season with oregano and combine with cooked pasta. Serve hot.
