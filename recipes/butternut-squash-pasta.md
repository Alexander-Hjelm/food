# Butternut Squash Pasta

> It's good shit.

## Tags

- pasta, main course, vegetarian

## Ingredients

| Ingredient         | Amount          | Prep           | Optional |
| -----------------  | --------------- | -------------- | -------- |
| Butternut squash   | 1/2             | peeled, seeded, diced |          |
| Marjoram           | to taste        |                |          |
| Thyme              | to taste        |                |          |
| Garlic             | 2-3 cloves      | minced         |          |
| Yellow onion       | 1               | diced          |          |
| Aromat             | to taste        |                | Optional |
| Salt               | 1 tsp           |                |          |
| Pepper             | to taste        |                |          |
| Water              | 200ml           |                |          |
| Olive oil          | 2 tbsp          |                |          |
| Pasta              | 250g            |                |          |
| Cheese             | to serve        | grated         | Optional |

## Instructions

1. Sauté squash, onion, garlic, marjoram, thyme, salt, pepper, and Aromat in olive oil until soft.  
2. Remove vegetables and purée in a blender until smooth.  
3. Cook pasta in the remaining water, then toss with the squash purée. Serve with grated cheese.
