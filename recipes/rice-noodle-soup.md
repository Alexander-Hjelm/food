# Rice Noodle Soup

> ?

## Tags

- asian, soup, vegetarian option, main course, quick meal

## Ingredients

| Ingredient         | Amount      | Prep                    | Optional |
| ----------------- | ----------- | ---------------------- | -------- |
| Water              | 1 L         |                        |          |
| Stock cube/bouillon| 1–2         |                        |          |
| Rice noodles (thin)| 200 g       |                        |          |
| Tofu               | 150 g       | thinly sliced          | Optional |
| Egg                | 1–2         | cooked and sliced thin |          |
| Hot pot sauce      | 2–3 tbsp    |                        | Optional |
| Bacon              | 50 g        | chopped                | Optional |
| Pak choi           | 1–2 heads   | whole, rinsed          |          |
| Spring onion       | 2 stalks    | sliced on bias         | Optional |

## Instructions

1. Bring water, bouillon and hot pot sauce to a boil to make the broth.
2. Cook the thin rice noodles according to package instructions, adding tofu slices in the last minute of cooking.
3. In a separate pan, fry the egg over high heat, then slice thinly and fold in half.
4. Add bacon to the broth if using, and cook until slightly crisp.
5. Add whole rinsed pak choi to the soup, letting it soften slightly.
6. Slice the spring onion on a diagonal for garnish.
7. Assemble the soup by arranging noodles, tofu, pak choi, egg slices, and spring onion in bowls.
