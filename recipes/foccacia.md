## Ingredients

| Ingredient          | Amount           | Prep       | Optional |
| ------------------- | ---------------- | ---------- | -------- |
| All-purpose flour    | 600g             |            |          |
| Yeast               | 25g              |            |          |
| Water               | 300ml            | lukewarm   |          |
| Sugar               | 1 tsp            |            |          |
| Olive oil           | 3 tbsp + for topping |          |          |
| Salt                | 1/2 tsp          |            |          |
| Fresh basil         | 30g              | chopped    |          |
| Cherry tomatoes     | 1 large handful  | halved     |          |
| White wine vinegar  | 1 tsp            |            | Optional |
| Garlic              | 1 clove          | minced     | Optional |
| Thyme               | 1 tsp            |            | Optional |
| Rosemary            | 1 tsp            | chopped    | Optional |

## Instructions

1. In a bowl, mix the dry ingredients: flour, sugar, and salt in the center. Prepare the yeast in lukewarm water and add to the dry ingredients.
2. Mix and knead the dough for at least 5 minutes until smooth.
3. Place the dough in an oiled bowl, cover, and let it rise for 40 minutes.
4. Roll out the dough in an oiled baking tray. Drizzle with olive oil and create small dimples with your fingers.
5. Add your toppings: cherry tomatoes, basil, garlic, thyme, and rosemary.
6. Cover with a damp cloth and let it rise for another 40 minutes.
7. Preheat the oven to 200Â°C and bake for 20 minutes.
8. Immediately brush with olive oil after baking and let cool.
9. Slice and serve with extra fresh basil on top.
