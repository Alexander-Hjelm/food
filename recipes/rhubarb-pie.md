# Rhubarb Pie

> ?

## Tags

## Ingredients

| Ingredient       | Amount      | Prep         | Optional |
| ---------------- | ----------- | ------------ | -------- |
| Butter           | 150 g       |              |          |
| Rolled oats      | 2 dl        |              |          |
| Wheat flour      | 2 dl        |              |          |
| Sugar            | pinch       |              |          |
| Rhubarb          | 500 g       | chopped      |          |
| Potato starch    | 1 tbsp      |              |          |
| Sugar            | pinch       |              |          |
| Cinnamon         | 1 tsp       |              | Optional |

## Instructions

1. Preheat the oven to 225°C.
2. Grease and flour a baking dish.
3. Prepare the filling by mixing chopped rhubarb, potato starch, sugar, and cinnamon.
4. Spread the filling evenly in the prepared dish.
5. Prepare the crumble topping by mixing margarine, rolled oats, flour, and a pinch of sugar until crumbly.
6. Distribute the crumble topping over the rhubarb filling.
7. Pre-bake the pie in the oven for 5 minutes at 225°C.
8. Continue baking in the lower part of the oven for 20 minutes, until the topping is golden and crisp.
9. Let cool slightly before serving.
