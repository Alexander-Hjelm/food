# Mapo Tofu (Semi-Vegetarian)

> Your chinese stepmom has just challenged you to a live cook-off. Perhaps this recipe can help you win, and thus normalize the tensions between your two countries? Or perhaps you are screwed and doomed to hear her constantly nag you about babies, marriages and stable government jobs for the rest of your life?

## Tags

- chinese, asian, main course, vegetarian

## Ingredients

### For frying

| Ingredient                       | Amount    | Prep      | Optional |
| -------------------------------- | --------- | ----------| -------- |
| Neutral oil                      | 1 tbsp    |           |          |
| Garlic                           | 4 cloves  | minced    |          |
| Ginger                           | 1 tbsp    | minced    |          |
| Sichuan peppercorns              | 1 tsp     | crushed   |          |
| Ground beef / Sausage / Soy meat | 200 g     |           |          |
| Fermented bean paste             | 1.5 tbsp  |           |          |
| Toasted chili oil/sauce          | 1 tbsp    |           |          |
| Soft tofu                        | 400 g     | diced     |          |
| Fresh coriander                  | to taste  | sliced    | Optional |
| Spring onion or leek             | 1 stalk   | sliced    |          |

### Mapu tofu sauce

| Ingredient                       | Amount    | Prep      | Optional |
| -------------------------------- | --------- | ----------| -------- |
| Chicken stock                    | 2 dl      |           |          |
| Oyster sauce                     | 1/2 tbsp  |           |          |
| Soy sauce                        | 2 tbsp    |           |          |
| Sesame oil                       | 0.5 tsp   |           |          |
| Corn starch                      | 1 tbsp    |           |          |
| Sugar                            | 1 tsp     |           |          |

## Instructions

1. Mix together the mapu tofu sauce ingredients in a glass
1. Sauté garlic, ginger for 30s
1. Add the meat and fry according to the choice of meat
1. Add Sichuan pepper, Bean paste and Chili oil and fry for another 30s
1. Add tofu and simmer for a few minutes
1. Add the mapu tofu sauce and simmer for a few minutes
1. Top with spring onions and coriander (if desired)
