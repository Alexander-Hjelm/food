# How to: Toast

> Because some country ass idiot was so uncreative with his breakfast that he only ate his sandwiches with Autosol™ and shoe polish. This document of the **sandwich spreads** that Alexander likes will prevent such accidents from happening again.

## Tags

- snack, breakfast, guide, vegetarian

## Ingredients

| Ingredient                 | Amount | Prep    | Optional |
| -------------------------- | ------ | ------- | -------- |
| Bread (toast, julvört, baguette, or homemade) | 2 slices |        |          |
| Butter                      | as needed |        |          |
| Margarine                   | as needed |        | Optional |
| Peanut butter               | as needed |        | Optional |
| Hummus                      | as needed |        | Optional |
| Tomato paste                | as needed |        | Optional |
| Mayonnaise                  | as needed |        | Optional |
| Cheese                      | as needed | sliced |          |
| Bell pepper                  | as needed | sliced | Optional |
| Cucumber                    | as needed | sliced | Optional |
| Banana                      | as needed | sliced | Optional |
| Lettuce                      | as needed |        | Optional |
| Spinach                     | as needed |        | Optional |
| Egg                         | 1–2     | cooked  | Optional |
| Heinz white beans in tomato sauce | as needed |     | Optional |

## Instructions

1. Choose your bread base.
2. Spread butter, margarine, or preferred spread.
3. Add toppings: basic (cheese, veggies, banana) or advanced (egg, beans).
4. Serve and enjoy a non-tragic breakfast.
