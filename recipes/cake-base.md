## Ingredients

| Ingredient   | Amount                 | Prep | Optional |
| ------------ | ---------------------- | ---- | -------- |
| Egg          | 5                      |      |          |
| Sugar        | 2,5 dl                 |      |          |
| Flour        | 75 g                   |      |          |
| Potato flour | 10 g                   |      |          |
| Baking soda  | 1,25 tsp               |      |          |
| Butter       | For buttering the tray |      |          |

## Instructions

1. Mix eggs and sugar with an electric whisk for ~3 minutes
2. Mix the dry ingredients in a separate bowl
3. Sift the dry mix into the wet mix
4. Mix to an even batter
5. Bread and butter the tray
6. Pour the batter into the tray
7. Bake for 40 minutes at 175 degrees C
