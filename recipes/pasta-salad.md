## Ingredients

| Ingredient       | Amount | Prep    | Optional |
| ---------------- | ------ | ------- | -------- |
| Pasta (any shape) | 200 g | cooked  |          |
| Iceberg lettuce   | 50 g  | chopped |          |
| Spinach           | 30 g  | chopped | Optional |
| Tomatoes          | 1     | diced   |          |
| Cherry tomatoes   | 6     | halved  | Optional |
| Cucumber          | 50 g  | diced   |          |
| Zucchini          | 50 g  | diced   | Optional |
| Red onion         | 30 g  | sliced  | Optional |
| Bell pepper       | 50 g  | diced   |          |
| Olives            | 30 g  | sliced  | Optional |

## Instructions

1. Cook pasta and let it cool.
2. Chop vegetables and combine with pasta.
3. Add dressing or vinaigrette and toss before serving.
