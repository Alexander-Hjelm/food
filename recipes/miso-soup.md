# Miso Soup

> ?

## Tags

- japanese, vegan, soup, main course

## Ingredients

| Ingredient    | Amount      | Prep        | Optional |
| ------------- | ----------- | ----------- | -------- |
| Kelp          | 10g         | soaked 15 min, trimmed and sliced |          |
| Water         | 1L          |             |          |
| Spring onions | 2-3 stalks  | chopped     |          |
| Tofu          | 150g        | cubed       | Optional |
| Miso paste    | 2-3 tbsp    |             |          |
| Salt          | 2 tbsp      |             |          |

## Instructions

1. Soak kelp for 15 minutes, trim stem, roll and slice into long strips.  
2. Heat kelp in water over medium heat; add tofu if using.  
3. Stir in miso paste and salt, bring to a gentle boil.  
4. Pour into bowls and garnish with chopped spring onions.
