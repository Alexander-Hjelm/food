# Roasted Almonds

> ?

## Tags

- snack, dessert, sweet, nuts

## Ingredients

| Ingredient    | Amount      | Prep     | Optional |
| ------------- | ----------- | -------- | -------- |
| Almonds       | 200g        |          |          |
| Sugar         | 1.5 dl      |          |          |
| Water         | 0.5 dl      |          |          |

## Instructions

1. Roast the almonds in a dry frying pan over medium heat until lightly golden.  
2. In a small saucepan, combine sugar and water and bring to a boil.  
3. Add the roasted almonds and cook while stirring until the sugar caramelizes and you hear cracking sounds.  
4. Remove from heat and let cool on parchment paper before serving.
