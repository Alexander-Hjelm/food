# Pesto Panini

> For real jabronis.

## Tags

- italian, sandwich, snack, vegetarian

## Ingredients

| Ingredient       | Amount       | Prep         | Optional |
| ---------------- | ------------ | ------------ | -------- |
| Fresh-baked loaf | 1            | sliced       |          |
| Mozzarella       | small pack   | shredded     |          |
| Basil pesto      | small jar    |              |          |
| Yellow onion     | 1            | sliced       |          |
| Tomato           | 1            | sliced       |          |
| Olive oil        | 1–2 tsp      |              |          |

## Instructions

1. Slice tomato and onion; cut bread in half. Spread pesto on both halves.
2. Layer shredded mozzarella on each side, then add onion on one half and tomato on the other. Close the sandwich.
3. Drizzle olive oil on top and bake at 180 °C for 10–15 minutes. Slice and serve.
