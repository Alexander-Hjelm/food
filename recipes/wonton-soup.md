# Wonton Soup

> ?

## Tags

- asian, chinese, soup

## Ingredients

### For the dumplings

| Ingredient                | Amount    | Prep     | Optional |
| ------------------------- | --------- | -------- | -------- |
| Minced chicken / Soy meat | 200 g     |          |          |
| Chinese chives            | to taste  |          |          |
| Wonton wrappers           | 30-40     |          |          |
| Light soy sauce           | 1 tbsp    |          |          |
| Egg                       | 1         |          |          |
| Oyster sauce              | 1/2 tbsp  |          |          |
| White pepper              | 1 tsp     |          |          |
| Salt                      | 1 tsp     |          |          |
| Sesame oil                | to taste  |          |          |
| Shaoxing wine             | to taste  |          | Optional |

### For the soup

| Ingredient                | Amount    | Prep     | Optional |
| ------------------------- | --------- | -------- | -------- |
| Chicken stock             | 1.5l      |          |          |
| White pepper              | to taste  |          |          |
| Salt                      | to taste  |          |          |
| Sesame oil                | to taste  |          |          |
| Seaweed                   | to taste  |          | Optional |
| Chili oil                 | to taste  |          | Optional |


## Instructions

1. Combine filling ingradients and let marinate for 10 minutes  
2. Wrap the dumplings
3. Mix the soup ingredients in a pot and bring to a boil
4. Carefully add in the dumplings to the soup and let boil until the dumplings float to the top
5. Ready to serve
