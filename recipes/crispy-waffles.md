# Crispy Waffles

> ?

## Tags

- swedish, breakfast, snack, dessert

## Ingredients

| Ingredient    | Amount      | Prep     | Optional |
| ------------- | ----------- | -------- | -------- |
| Butter        | 200g        | melted   |          |
| Water         | 500ml       |          |          |
| Milk          | 400ml       |          |          |
| Wheat flour   | 7 dl        |          |          |
| Baking powder | 4 tsp       |          |          |

## Instructions

1. Mix melted butter, water, and milk in a bowl.  
2. Add flour and baking powder, stir until smooth.  
3. Preheat waffle iron and cook batter until golden and crispy.
