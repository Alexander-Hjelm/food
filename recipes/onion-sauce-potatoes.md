# Onion Sauce for Potatoes

> ?

## Tags

- swedish, sauce, side dish, comfort food

## Ingredients

| Ingredient        | Amount      | Prep         | Optional |
| ----------------- | ----------- | ------------ | -------- |
| Cooking cream      | 200 ml      |              |          |
| Wheat flour        | 3 tbsp      |              |          |
| Onion              | 3           | sliced       |          |
| White pepper       | to taste    |              |          |
| Salt               | to taste    |              |          |
| Stock cube         | 0.5         | crumbled     |          |
| Brown beans        | to serve    | cooked       | Optional |
| Potatoes           | to serve    | boiled       | Optional |
| Pork               | to serve    | fried        | Optional |

## Instructions

1. Slice the onions thinly.
2. In a saucepan, gently saut√© the onions until soft and translucent.
3. Sprinkle in the flour and stir to coat the onions evenly.
4. Gradually add the cooking cream while stirring continuously to avoid lumps.
5. Crumble in half a stock cube and season with salt and white pepper to taste.
6. Simmer the sauce gently until thickened and flavorful.
7. Serve warm alongside boiled potatoes, brown beans, and fried pork if desired.
