# Noodles

> That's what she said.

## Tags

- asian, pasta, main course, vegetarian

## Ingredients

| Ingredient    | Amount      | Prep     | Optional |
| ------------- | ----------- | -------- | -------- |
| Wheat flour   | 500g        |          |          |
| Water         | 200ml       |          |          |
| Egg           | 1-2         | beaten   |          |
| Salt          | 1 tsp       |          |          |

## Instructions

1. Mix flour and salt in a bowl.  
2. Gradually add water and beaten egg, knead into a smooth dough.  
3. Roll out dough thinly and cut into desired noodle shape. Cook in boiling water until tender.
