# Yogurt Sauce

> Don't kiss your girlfriend, mom or dog after having this. I am speaking from several bad personal experiences here.

## Tags

- sauce, dressing, mediterranean, condiment

## Ingredients

| Ingredient    | Amount      | Prep     | Optional |
| ------------- | ----------- | -------- | -------- |
| Yoghurt       | 200ml       |          |          |
| Dill          | 2 tbsp      | chopped  |          |
| Chives        | 2 tbsp      | chopped  |          |
| Red onion     | 2 tbsp      | diced    |          |
| Vinegar       | 1 tsp       |          |          |
| Lemon juice   | 1 tsp       |          |          |
| Garlic        | 1 clove     | minced   |          |
| Olive oil     | 1 tbsp      |          |          |

## Instructions

1. Add yoghurt to a bowl.  
2. Stir in dill, chives, and diced red onion.
