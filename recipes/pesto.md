# Pesto

> ?

## Tags

- italian, sauce, condiment, vegetarian

## Ingredients

| Ingredient       | Amount      | Prep       | Optional |
| ---------------- | ----------- | ---------- | -------- |
| Fresh basil      | 50g         |            |          |
| Olive oil        | 72g         |            |          |
| Parmesan cheese  | 40g         | grated     |          |
| Pine nuts        | 1 tbsp      |            |          |
| Garlic cloves    | 2           | crushed    |          |
| Salt             | 0.25 tsp    |            |          |

## Instructions

1. Combine basil, pine nuts, and garlic in a food processor.  
2. Slowly blend in olive oil until smooth.  
3. Mix in grated parmesan and salt, then blend briefly to combine.
