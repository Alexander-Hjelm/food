# Shakshuka

> ?

## Tags

- middle eastern, vegetarian, main course

## Ingredients

| Ingredient         | Amount      | Prep     | Optional |
| ------------------ | ----------- | -------- | -------- |
| Garlic             | 2 cloves    | minced   |          |
| Onion              | 1 medium    | diced    |          |
| Bell pepper        | 1 large     | diced    |          |
| Zucchini           | 1 small     | diced    |          |
| Celery             | 1 stalk     | sliced   |          |
| Crushed tomatoes   | 400ml       |          |          |
| Italian seasoning  | 1 tsp       |          |          |
| Mint               | 1 tbsp      | chopped  | Optional |
| Parsley            | 1 tbsp      | chopped  | Optional |
| Chili paste        | 1–2 tsp     |          |          |
| Salt               | To taste    |          |          |
| Pepper             | To taste    |          |          |
| Eggs               | 3–4         | poached  |          |
| Cheese             | 30g         | crumbled | Optional |

## Instructions

1. Sauté garlic, onion, bell pepper, zucchini, and celery in a pan until softened.
2. Add crushed tomatoes, Italian seasoning, chili paste, salt, and pepper. Let the mixture simmer for a few minutes.
3. Stir in mint and parsley.
4. Create small wells in the sauce and gently add the eggs.
5. Cover with a lid and cook until the eggs are poached to your liking.
6. Top with cheese before serving.
