# Raisin Cookies

> ?

## Tags

- dessert, baking, cookies, snack, sweet

## Ingredients

| Ingredient         | Amount       | Prep                  | Optional |
| ------------------ | ------------ | -------------------- | -------- |
| Butter             | 120 g        | melted               |          |
| Wheat flour        | 340 g        |                      |          |
| Raisins            | 60 g         | chopped              |          |
| Milk               | a splash     |                      | Optional |
| Eggs               | 3            | beaten               |          |
| Honey              | 2–3 tbsp     |                      | Optional |

## Instructions

1. In a bowl, mix the flour and chopped raisins.
2. Add the beaten eggs and 2–3 tablespoons of honey.
3. Melt the butter and pour it into the mixture.
4. Mix thoroughly and roll the dough into a log. Knead lightly and wrap in plastic wrap.
5. Chill in the refrigerator for 1 hour.
6. Preheat the oven to 200°C.
7. Slice the dough log into cookie portions and place them on a baking tray.
8. Bake for 10–15 minutes until lightly golden.
9. Let cool slightly before serving.
