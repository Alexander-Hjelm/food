## Ingredients

| Ingredient      | Amount    | Prep        | Optional |
| --------------- | -------- | ----------- | -------- |
| Flour           | 600g     |            |          |
| Powdered sugar  | 100g      |            |          |
| Eggs            | 4        |            |          |
| Butter          | 160g     | melted     |          |
| Baking powder   | 5 tsp    |            |          |
| Salt            | 3 tsp    |            |          |
| Egg             | 1        | beaten (for brushing) | Optional |

## Instructions

1. Preheat the oven to 180Â°C.  
2. In a bowl, mix flour, powdered sugar, baking powder, and salt.  
3. Add melted butter and 4 eggs, and combine into a dough.  
4. Chill the dough in the fridge for 10 minutes.  
5. Roll out the dough to 1 cm thickness and cut into desired shapes.  
6. Brush the tops with beaten egg.  
7. Bake for 10 minutes or until golden brown. Serve warm.
