# Hot Pockets

## Ingredients

| Ingredient        | Amount       | Prep         | Optional |
| ----------------- | -----------  | ------------ | -------- |
| Eggs              | 6            | beaten       |          |
| Soy mince         | 120g         |              |          |
| Shredded cheese   | To taste     |              |          |
| Flour             | 400g         |              |          |
| Baking powder     | 2 tsp        |              |          |
| Greek yogurt      | 420g         |              |          |
| Onion powder      | To taste     |              |          |
| Garlic powder     | To taste     |              |          |
| Salt              | To taste     |              |          |
| Olive oil         | For brushing |              |          |

## Instructions

1. Preheat oven to 175Â°C.  
2. Prepare the dough: mix flour, baking powder, Greek yogurt, onion powder, garlic powder, and salt until combined. Divide into 8 equal portions.  
3. Prepare the filling: beat the eggs, mix in soy mince and shredded cheese.  
4. Roll out each dough portion, place filling in the center, and fold to seal into pockets.  
5. Brush pockets with olive oil and place on a baking tray.  
6. Bake for 15 minutes until golden. Serve warm.
