## Ingredients

| Ingredient                       | Amount   | Prep     | Optional |
| -------------------------------- | -------- | -------- | -------- |
| Eggs                             | 3        | whisked  |          |
| Tomatoes                         | 3        | diced    |          |
| Garlic                           | 1 tbsp   | minced   |          |
| Tomato paste                     | 2 tbsp   |          |          |
| Sugar                            | 1 tsp    |          |          |
| Salt                             | 0.5 tsp  |          |          |
| Broth                            | 1.2 L    |          |          |
| Noodles                          | 1 portion|          |          |
| Spinach or pak choi              | 1 handful|    | Optional |
| Soy sauce                        | 1 tbsp   |          |          |
| Chinese fermented vegetables     | to taste |     | Optional |

## Instructions

1. Cook whisked eggs into a thin omelette; break into pieces.
2. Saut√© garlic, tomato paste, and tomatoes briefly.
3. Add broth, sugar, salt, soy sauce, noodles, and greens; simmer until noodles are done.
4. Add egg pieces and top with fermented vegetables.
