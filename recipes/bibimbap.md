# Bibimbap

> ?

## Tags

- korean, asian, main course

## Ingredients

| Ingredient        | Amount | Prep                 | Optional |
| ----------------- | ------ | -------------------- | -------- |
| Rice              | 300 g  | cooked               |          |
| Egg               | 1      | fried                |          |
| Carrot            | 1      | julienned            |          |
| Cucumber          | 1      | julienned            |          |
| Spinach           | 150 g  |                      |          |
| Mushrooms         | 150 g  | sliced               | Optional |
| Bean sprouts      | 100 g  |                      | Optional |
| Soy sauce         | to taste |                    |          |
| Sesame oil        | to taste |                    |          |
| Garlic            | 2 tsp  | minced               |          |
| Neutral oil       | as needed |                  |          |

### Gochujang Sauce

| Ingredient        | Amount | Prep | Optional |
| ----------------- | ------ | ---- | -------- |
| Gochujang         | 100 ml |      |          |
| Sesame oil        | 50 ml  |      |          |
| Water             | 50 ml  |      |          |
| Sugar             | 60 ml  |      |          |
| Sesame seeds      | 50 ml  | toasted |       |
| Rice vinegar      | 12 ml  |      |          |
| Garlic            | 2 tsp  | minced |         |

### Fermented White Cabbage (Optional)

| Ingredient        | Amount | Prep        | Optional |
| ----------------- | ------ | ----------- | -------- |
| White cabbage     | 500 g  | shredded    | Optional |
| Salt              | 3–4 tbsp |           | Optional |
| Caraway seeds     | 1 tsp  |             | Optional |
| Juniper berries   | 4–5    | crushed     | Optional |

## Instructions

1. Cook the rice and keep warm.
2. Mix all sauce ingredients until smooth and set aside.
3. Fry carrots and cucumber strips in neutral oil with soy sauce and garlic.
4. Fry spinach quickly in sesame oil, soy sauce, and garlic.
5. Fry mushrooms in sesame oil and soy sauce if using.
6. Fry an egg sunny-side up.
7. (Optional cabbage) Salt shredded cabbage heavily and let sit for 30 minutes, pressing occasionally. Rinse, mix with spices, pack into a jar, and ferment for 1–2 weeks, releasing pressure daily.
8. Assemble rice in a bowl, arrange vegetables neatly on top.
9. Add fried egg and a generous spoon of sauce.
10. Mix well before eating and enjoy.
