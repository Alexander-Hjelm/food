# Pizza Dough (1 Pizza)

> ?

## Tags

## Ingredients

| Ingredient      | Amount       | Prep     | Optional |
| --------------- | ----------- | -------- | -------- |
| Water           | 2.5 dl      |          |          |
| Yeast           | 25 g        |          |          |
| Olive oil       | 2 tbsp      |          |          |
| Salt            | To taste    |          |          |
| Wheat flour     | 6.5 dl      |          |          |
| Wheat gluten    | 1 dl        |          |          |
| Sugar           | 2 tsp       |          |          |

## Instructions

1. Warm the water slightly and dissolve the yeast in it.  
2. Add olive oil, salt, sugar, wheat flour, and wheat gluten. Knead into a smooth dough.  
3. Let the dough rise for about 15 minutes.  
4. Roll out the dough to desired shape and thickness.  
5. Preheat oven to 225Â°C and bake the pizza base for 15 minutes before adding toppings.
