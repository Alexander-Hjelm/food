# Aloo Paratha

> Hop in, boys — we're heading to Bengal!

## Tags

- indian, main course, flatbread, vegetarian

| Ingredient        | Amount      | Prep          | Optional |
| ----------------- | ----------- | ------------- | -------- |
| Flour             | —           |               |          |
| Potatoes          | —           | boiled, mashed|          |
| Cooking oil       | —           |               |          |
| Yoghurt/Crème fraîche | —       | for serving   | Optional |
| Black pepper      | to taste    | ground        |          |
| Cumin             | to taste    |               |          |
| Chili powder      | to taste    |               |          |
| Salt              | to taste    |               |          |

## Instructions

1. Boil and mash potatoes.
2. Toast pepper, cumin, and chili on low heat; mix into mashed potatoes with salt.
3. For the dough: combine 3 parts flour with 1.5 parts potato, add oil and salt
4. Mix in the mashed potatoes with the dough.
5. Knead into a soft dough; rest 30 minutes. Keep massaging with oil to prevent the dough from drying out.
6. Form dough balls, roll out, and pan-fry in oil over medium-low heat.
7. Serve with yoghurt or crème fraîche.
