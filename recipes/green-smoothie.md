# Green Smoothie

> What 15% of all hipsters are made of.

## Tags

- smoothie, drink, breakfast, snack, vegetarian

## Ingredients

| Ingredient        | Amount | Prep        | Optional |
| ----------------- | ------ | ----------- | -------- |
| Green grapes      | 150 g  |             |          |
| Spinach           | 150 g  | finely chopped, half-thawed |          |
| Banana            | 1      |             |          |
| Milk / Coconut milk | 0.5 dl |             |          |

## Instructions

1. Combine grapes, spinach, banana, and coconut milk.
2. Blend until smooth.
3. Drink slowly for your own safety.
