## Ingredients

| Ingredient    | Amount      | Prep           | Optional |
| ------------- | ----------- | -------------- | -------- |
| Apple         | 1-2         | diced          |          |
| Pear          | 1-2         | diced          |          |
| Banana        | 1-2         | diced          |          |
| Orange        | 1           | diced          |          |
| Mandarin      | 1-2         | diced          |          |
| Yogurt        | 100-200g    |                |          |
| Mayonnaise    | 1-2 tbsp    |                | Optional |

## Instructions

1. Dice all fruits into bite-sized pieces.  
2. In a mixing bowl, combine the diced fruits with yogurt and mayonnaise.  
3. Mix gently until everything is evenly coated.  
4. Serve immediately or chill briefly in the fridge before serving.
