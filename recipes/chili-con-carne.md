# Chili Con Carne

> The door opens on the rocket, the banana command comes out. Banana contact of the third degree!

## Tags

* main course, mexican, american

## Ingredients

| Ingredient                  | Amount     | Prep              | Optional |
| --------------------------- | ---------- | ----------------- | -------- |
| Minced meat / Soy meat      | 500 g      |                   |          |
| Crushed tomatoes            | 2 packages |                   |          |
| White beans in tomato sauce | 2 cans     |                   |          |
| Garlic                      | 5 cloves   | pressed / chopped |          |
| Yellow onion                | 2-3        | diced             |          |
| Broth cube                  | 1-2        |                   |          |
| Paprika powder              | 1 tbsp     |                   |          |
| Cumin powder                | 3/4 tbsp   |                   |          |
| Chili powder                | 3/4 tbsp   |                   |          |
| Salt                        | to taste   |                   |          |
| Black pepper                | to taste   |                   |          |
| Sambal oelek                | 1-2 tbsp   |                   | Optional |
| Chili sauce                 | to taste   |                   | Optional |

## Instructions

1. Saut√© garlic and onion in a work pan.
2. Add minced meat / soy meat to the pan and pre-fry the meat until brown.
3. Combine and add remaining ingredients.
4. Slow-cook for ~30 minutes or as long as desired.
