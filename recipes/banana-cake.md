## Ingredients

| Ingredient      | Amount | Prep          | Optional |
| --------------- | ------ | ------------- | -------- |
| Banana          | 3      | mashed        |          |
| Egg             | 2      | beatem        |          |
| Sugar           | 60g    |               |          |
| Flour           | 200g   |               |          |
| Baking powder   | 8g     |               |          |
| Butter          | 120g   |               |          |
| Yoghurt         | 50g    |               |          |
| Salt            | 2g     |               |          |
| Vanilla extract |        |               |          |
| Panko           |        |               |          |
| Cinnamon        |        |               | Optional |
| Nuts            |        | crushed       | Optional |

## Instructions

1. Combine all ingredients in a medium bowl.
2. Bake on 175 degrees for 40 min, then cover with tin foil and bake for another 30 minutes.
