# Thai Red Curry

> Warning: may induce uncontrollable cravings for rice and coconut milk.

## Tags

- thai, asian, main course, spicy

## Ingredients

| Ingredient        | Amount       | Prep         | Optional |
| ----------------- | ----------- | ------------ | -------- |
| Chicken pieces    | 500g        |             |          |
| Potatoes          | 2-3 medium  | peeled, diced |          |
| Carrots           | 2           | peeled, sliced |          |
| Onion             | 1           | thinly sliced |          |
| Bell pepper       | 1           | sliced       | Optional |
| Thai red curry paste | 6-7 tbsp | adjust for spice |          |
| Coconut milk      | 1/2 can     |             |          |
| Rice              | As needed   |             |          |
| Salt              | To taste    |             |          |
| Cumin             | To taste    |             |          |

## Instructions

1. Peel and dice potatoes and carrots.  
2. Fry chicken pieces in their own fat if possible.  
3. Combine chicken, potatoes, carrots, and other vegetables in a pot and cook for 20 minutes.  
4. After vegetables have cooked for 5 minutes, add 6–7 tablespoons of red curry paste, depending on desired heat.  
5. Cook rice according to package instructions.  
6. Thinly slice onion and bell pepper. Add onion to the curry.  
7. Add half a can of coconut milk.  
8. Add bell pepper toward the end of cooking.  
9. After the curry has simmered for 10–15 minutes, season with salt and cumin. Serve hot with rice.

Use **red** thaicurry paste for **chicken**
Use **green** thaicurry paste for **beef**
Use **yellow** thaicurry paste for **fish**
