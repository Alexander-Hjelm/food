# Klimpsoppa (Dumpling Soup)

> ?

## Tags

- swedish, soup, main course, comfort food

## Ingredients

| Ingredient         | Amount       | Prep                  | Optional |
| ----------------- | ------------ | ------------------- | -------- |
| Summer vegetables / root vegetables | 300g | diced                 |          |
| Onion              | 1 large      | chopped               |          |
| Parsley            | 2 tbsp       | fresh or dried       |          |
| Water              | 1.5 L        |                       |          |
| Vegetable stock cubes | 3          |                       |          |
| Salt               | To taste     |                       |          |
| Eggs               | 3            | beaten               |          |
| Flour              | 3.5 dl       |                       |          |
| Black pepper       | To taste     |                       |          |
| Grill spice        | 1 tsp        |                       | Optional |

## Instructions

1. Chop the vegetables and onion.  
2. In a large pot, bring water and stock cubes to a boil. Add vegetables, onion, parsley, and salt. Simmer until vegetables are tender.  
3. For the dumplings (klimp), mix eggs, flour, salt, black pepper, and grill spice into a smooth dough.  
4. Drop spoonfuls of the dumpling mixture into the simmering soup.  
5. Cook for 10â€“15 minutes until dumplings are cooked through.  
6. Serve hot, garnished with extra parsley if desired.
