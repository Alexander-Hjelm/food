# Edamame beans (steamed)

> Look at those massive eda-mamas, huehuehuehuehue!

## Tags

- snack, vegetarian, vegan, appetizer

## Ingredients

| Ingredient             | Amount      | Prep         | Optional |
| ---------------------- | ----------- | ------------ | -------- |
| Edamame beans (frozen) | 100g        |              |          |
| Salt                   | 1/2 tsp     |              |          |
| Chili powder           | 1/2 tsp     |              | Optional |

## Instructions

1. Steam the beans for 2-3 minutes.
2. Sprinkle the bean pods with salt and chili powder and serve.
