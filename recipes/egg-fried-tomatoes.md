# Egg Fried Tomatoes

> If you need further instructions, you should be forcibly castrated.

## Tags

- asian, main course, breakfast

## Ingredients

| Ingredient      | Amount       | Prep       | Optional |
| --------------- | ----------- | ---------- | -------- |
| Eggs            | 3           |            |          |
| Tomatoes        | 3-4         | chopped    |          |
| Salt            | to taste    |            |          |
| Sugar           | to taste    |            |          |
| Soy sauce       | to taste    |            |          |
| Spring onions   | 2-3         | sliced     | Optional |

## Instructions

1. Fry the eggs in a pan until lightly set.  
2. Fry the chopped tomatoes in the same pan until softened.  
3. Mix eggs and tomatoes together, season with salt, sugar, and soy sauce, then simmer briefly.  
4. Garnish with sliced spring onions and serve hot.
