# Calzones

> Puedo ver tus calzones, hombre!

## Tags

- italian, main course, snack

## Ingredients

| Ingredient       | Amount      | Prep       | Optional |
| ---------------- | ----------- | --------- | -------- |
| Wheat flour      | 400g        |           |          |
| Water            | 200ml       |           |          |
| Yeast            | 20g         |           |          |
| Pizza sauce      | 100ml       |           |          |
| Mozzarella       | 150g        | grated    |          |
| Baby spinach     | 50g         | chopped   |          |
| Bell pepper      | 1           | sliced    |          |
| Spring onions    | 2           | sliced    | Optional |

## Instructions

1. Make dough from flour, water, and yeast; let rise for 30 minutes.  
2. Prepare sauce by mixing pizza and pasta sauce if desired.  
3. Fill dough with mozzarella, spinach, bell pepper, and spring onions. Fold, seal, and bake until golden.
