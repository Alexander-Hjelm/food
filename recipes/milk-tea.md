# Milk Tea

> You said it, man. Nobody fucks with the Jesus.

## Tags

- drink, indian, beverage, tea

## Ingredients

| Ingredient  | Amount         | Prep  | Optional |
| ----------- | -------------- | ----- | -------- |
| Milk        | 3 parts        |       |          |
| Water       | 1 part         |       |          |
| Sugar       | 1.5 tsp per cup|       |          |
| Tea leaves  | 2 tsp per cup  |       |          |

## Instructions

1. Combine milk and water in a saucepan.
2. Stir in sugar and heat until boiling.
3. Add tea leaves and boil until dark brown, adjusting heat to prevent overflow.
4. Let rest a few minutes, then strain and serve.
