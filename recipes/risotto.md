# Risotto

> ?

## Tags

- italian, main course, vegetarian

## Ingredients

| Ingredient              | Amount        | Prep                  | Optional |
| ----------------------- | ------------- | --------------------- | -------- |
| Porridge rice (grötris) | 3 dl          | Soaked overnight      |          |
| Red onion               | 2 small       | Finely chopped        |          |
| Garlic                  | 20 ml         | Chopped               |          |
| Broth                   | 2 cubes, 9 dl | Dissolved in water    |          |
| Olive oil               | 2 tbsp        |                       |          |
| White wine vinegar      | 30 ml         | Mixed with 1 dl water |          |
| Cheese                  | 100 g         |                       |          |
| Butter                  | 25 g          |                       |          |
| Salt                    |               |                       |          |
| Pepper                  |               |                       |          |
| Mushrooms               |               |                       | Optional |

## Instructions

1. Sauté garlic and onion until colored nicely
2. Rinse and add rice to the pan. Fry on high heat for a few minutes
3. Add broth gradually while stirring
4. Leave on low heat until thick (Usually 15-20 minutes). Stir every now and then to prevent the rice from burning to the pot
5. Add cheese and butter, stir until cheese has completely melted and mixed in
6. Add salt and pepper to taste
7. Keep on heat for another 3-4 minutes, then serve
