# Gin and Tonic

> ?

## Tags

- cocktail, drink, alcoholic

## Ingredients

| Ingredient    | Amount      | Prep     | Optional |
| ------------- | ----------- | -------- | -------- |
| Gin           | 60ml        |          |          |
| Tonic water   | 180ml       |          |          |
| Lime juice    | 1 tsp       |          |          |
| Mint leaf     | 1 leaf      |          | Optional |
| Lime slice    | 1 slice     |          | Optional |

## Instructions

1. Fill a glass with ice.  
2. Pour gin and tonic water over the ice.  
3. Add lime juice, stir gently, and garnish with mint leaf and lime slice.
