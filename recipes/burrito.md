# Burrito

> Okay, now let's have some yum, boys!

## Tags

- mexican, main course

## Ingredients

| Ingredient        | Amount       | Prep       | Optional |
| ----------------- | -----------  | ---------- | -------- |
| Tortilla          | 1–2          |          |          |
| Salad             | 50–100 g     | chopped   |          |
| Sauce             | 2–3 tbsp     |          |          |
| Beans in tomato sauce | 150 g     |           |          |
| Rice              | 100 g        | cooked    |          |

## Instructions

1. Spread cooked rice evenly on the tortilla.
2. Add beans (drained), chopped salad, and sauce.
3. Fold the tortilla into a burrito and serve.
