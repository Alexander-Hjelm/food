# American Pancakes

> ?

## Tags

- american, breakfast, dessert

## Ingredients

| Ingredient      | Amount | Prep     | Optional |
| --------------- | ------ | -------- | -------- |
| Wheat flour     | 500 ml |          |          |
| Baking powder   | 2 tbsp |          |          |
| Sugar           | 4 tbsp |          |          |
| Salt            | 2 ml   |          |          |
| Milk            | 500 ml |          |          |
| Butter          | 4 tbsp | melted  |          |
| Eggs            | 2      |          |          |

## Instructions

1. Mix all dry ingredients in a bowl.
2. Add milk, melted butter, and eggs.
3. Stir until just combined.
4. Fry pancakes in a lightly greased pan over medium heat.
5. Cook until bubbles form, flip, and cook until golden.
6. Makes about 10 pancakes.
