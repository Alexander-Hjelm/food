# Meringues

> Fluffy, sweet, and dangerously easy to eat all at once.

## Tags

- dessert, baking, snack, french

## Ingredients

| Ingredient     | Amount | Prep | Optional |
| -------------- | ------ | ---- | -------- |
| Egg whites     | 100 g  |      |          |
| Sugar          | 70 g   |      |          |
| Lemon juice / vinegar / distilled vinegar | A few drops | |          |
| Vanilla aroma  | A few drops | |          |

## Instructions

1. Add the egg whites to a clean bowl and begin whisking.
2. When the mixture becomes foamy, add a few drops of lemon juice, vinegar, or distilled vinegar.
3. Continue whisking and gradually add the sugar.
4. Add a few drops of vanilla aroma.
5. Whisk until the mixture reaches a firm, glossy consistency.
6. Pipe or spoon the meringue onto a baking sheet lined with parchment paper.
7. Bake at **150Â°C for 30 minutes**.
8. Let cool completely before serving.
