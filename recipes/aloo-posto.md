# Aloo Posto

> Aloo-post-my-ass-to-Santa-Claus-and-give-him-a-smack-on-the-pork-belly

## Tags

- indian, main course, vegetarian

## Ingredients

| Ingredient       | Amount      | Prep       | Optional |
| ---------------- | ----------- | --------- | -------- |
| Poppy seeds      | 200g        | soaked overnight |          |
| Salt             | to taste    |           |          |
| Potatoes         | 3-4         | diced      |          |
| Turmeric         | 1 tsp       |           |          |
| Chili powder     | 1 tsp       |           | adjust to taste |
| Mustard oil      | 2 tbsp      |           |          |

## Instructions

1. Soak poppy seeds overnight. Blend with a pinch of salt until smooth, toothpaste-like consistency.  
2. Dice potatoes and fry in mustard oil until slightly soft.  
3. Add turmeric, chili powder, and cover with a lid while cooking.  
4. Mix in poppy seed paste with a little water. Cook covered until potatoes are tender.  
5. Serve hot with rice and daal.
