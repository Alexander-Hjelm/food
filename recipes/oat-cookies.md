## Ingredients

| Ingredient        | Amount       | Prep         | Optional          |
| ----------------- | -----------  | ------------ | ---------------- |
| Unsalted butter   | 227g         | softened     |                  |
| White sugar       | 150g         |              |                  |
| Brown sugar       | 220g         | packed       |                  |
| Eggs              | 2            |              |                  |
| Vanilla extract   | 2 tsp        |              |                  |
| Flour             | 250g         |              |                  |
| Baking soda       | 1 tsp        |              |                  |
| Salt              | 1 tsp        |              |                  |
| Ground cinnamon   | 2 tsp        |              |                  |
| Rolled oats       | 240g         |              |                  |
| Ground cardamom   | To taste     |              | Optional         |
| Ground nutmeg     | To taste     |              | Optional         |

## Instructions

1. Preheat oven to 190Â°C.  
2. Cream together butter, white sugar, and brown sugar until smooth.  
3. Beat in eggs and vanilla extract.  
4. In a separate bowl, mix flour, baking soda, salt, cinnamon, and optional spices.  
5. Gradually combine dry ingredients with wet mixture, then fold in oats.  
6. Drop spoonfuls of dough onto a baking sheet.  
7. Bake for 10 minutes until edges are lightly golden. Cool slightly before serving.
