# Hummus

> Dinner is served, pendejos!!

## Tags

- middle eastern, snack, dip

## Ingredients

| Ingredient    | Amount      | Prep     | Optional |
| ------------- | ----------- | -------- | -------- |
| Chickpeas     | 400g        | drained  |          |
| Tahini        | 60g         |          |          |
| Garlic        | 2 cloves    | pressed  |          |
| Salt          | 1 tsp       |          |          |
| Olive oil     | 2 tbsp      |          |          |
| Lemon juice   | 2 tbsp      |          |          |

## Instructions

1. Blend chickpeas, tahini, garlic, salt, and lemon juice in a food processor until smooth.  
2. Slowly add olive oil while blending to achieve a creamy consistency.  
3. Serve with a drizzle of olive oil and optional garnish like paprika or parsley.
