# Tuna Salad Wraps

> The wrap that saves lunch when the fridge looks empty.

## Tags

- american, lunch, snack, quick meal, wraps

## Ingredients

| Ingredient      | Amount      | Prep     | Optional |
| --------------- | ----------- | -------- | -------- |
| Tuna in oil     | 200g        | drained  |          |
| Mayonnaise      | 3 tbsp      |          |          |
| Boiled eggs     | 2           | chopped  |          |
| Lettuce         | 4 leaves    | shredded |          |
| Tomatoes        | 2           | diced    |          |
| Red onion       | 1/4         | diced    |          |
| Tortillas       | 4           |          |          |
| Salt            | To taste    |          |          |
| Pepper          | To taste    |          |          |

## Instructions

1. Drain the tuna and mix it in a bowl with mayonnaise, chopped boiled eggs, salt, and pepper.  
2. Add diced tomatoes, diced red onion, and shredded lettuce. Stir gently to combine.  
3. Lay out the tortillas and divide the tuna mixture evenly among them.  
4. Roll the tortillas into wraps and serve immediately, or chill for later.
