# Broccoli Soup

> Klart för start! Rulla ut och rulla på!

## Tags

- soup, vegetarian, main course, swedish

## Ingredients

| Ingredient        | Amount        | Prep         | Optional |
| ----------------- | ------------- | ------------ | -------- |
| Onion             | 1             | diced        |          |
| Potatoes          | 3-4 (≈300g)   | peeled, diced|          |
| Broccoli          | 250g          | chopped      |          |
| Vegetable broth   | 900ml         |              |          |
| Cooking cream     | 250ml         |              |          |
| Salt              | to taste      |              |          |
| Pepper            | to taste      |              |          |

## Instructions

1. Boil onion, potatoes, and broccoli in vegetable broth for 10 minutes.  
2. Purée the vegetables until smooth.  
3. Add cooking cream, bring to a simmer, and season with salt and pepper.
