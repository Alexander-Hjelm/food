# Re Gan Mian

> ?

## Tags

- chinese, asian, noodle, main course, street food

## Ingredients

| Ingredient        | Amount     | Prep     | Optional |
| ----------------- | ---------- | -------- | -------- |
| Wheat noodles     | 400 g      | cooked   |          |
| Soy sauce         | 2 tbsp     |          |          |
| Chili oil         | 1–2 tbsp   |          |          |
| Cumin powder      | 1 tsp      |          |          |
| Sesame seeds      | 1 tbsp     | toasted  |          |
| Salt              | to taste   |          |          |
| Black pepper      | to taste   | ground   |          |
| Yellow bean sauce | 1–2 tbsp   |          |          |

## Instructions

1. Cook the noodles until just tender, then drain well.
2. In a bowl, mix the noodles with soy sauce, chili oil, cumin powder, sesame seeds, salt, and pepper.
3. Add the yellow bean sauce and toss thoroughly until every strand is coated.
4. Serve immediately, optionally topped with extra sesame seeds or chili oil.
