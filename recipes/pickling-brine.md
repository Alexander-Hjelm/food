# Pickling Brine

> ?

## Tags

- guide, condiment, scandinavian, swedish

## Ingredients

| Ingredient        | Amount | Prep        | Optional |
| ----------------- | ------ | ----------- | -------- |
| Water             | 1 l    |             |          |
| Vinegar (12%)     | 200 ml |             |          |
| Garlic            | 3 cloves | sliced   |          |
| Mustard seeds     | 2 tbsp |             |          |
| Dill seeds        | 2 tbsp |             |          |
| Salt              | 60 ml  |             |          |
| Sugar             | 2 tbsp |             |          |

## Instructions

1. Combine all ingredients in a pot.
2. Bring the mixture to a boil.
3. Remove from heat.
4. Pour the hot brine into clean jars together with the vegetables or items to be pickled.
5. Seal and let cool before storing in the refrigerator.
