# Rice Porridge (Risgrynsgröt)

> ?

## Tags

- swedish, breakfast, main course, dessert

## Ingredients

| Ingredient              | Amount   | Prep             | Optional |
| ----------------------- | -------- | ---------------- | -------- |
| Porridge rice (grötris) | 3 dl     | Soaked overnight |          |
| Milk                    | 10 dl    |                  |          |
| Sugar                   | 20 ml    |                  |          |
| Cinnamon                | To taste |                  | Optional |

## Instructions

1. Rinse the rice grains
2. Mix rice grains and milk into a pot and heat up until boiling
3. Leave on low heat until thick (Usually 20 minutes). Stir every now and then to prevent the porridge from burning to the pot
4. Add sugar and mix
5. (Optional) Serve with cinnamon powder and additional sugar as desired
