# Cookies with condensed milk

> 

## Tags

- dessert, baking, cookies, snack, sweet

## Ingredients

| Ingredient         | Amount       | Prep                  | Optional |
| ------------------ | ------------ | -------------------- | -------- |
| Butter             | 70 g         | melted                |          |
| Sugar              | 10 g         |                      |          |
| Salt               | 1 g          |                      |          |
| Condensed milk     | 50 g         |                      |          |
| Wheat flour        | 150 g        |                      |          |
| Cornmeal           | 15 g         |                      |          |
| Optional flavors   | as desired   | cocoa powder, matcha, Nutella | Optional |

## Instructions

1. Melt the butter and mix it with sugar, salt, and condensed milk.
2. Add wheat flour and cornmeal, mixing until a smooth dough forms.
3. If desired, add cocoa powder, matcha, or a swirl of Nutella for flavor.
4. Chill the dough in the refrigerator for 30 minutes.
5. Shape the dough into small cookie portions.
6. Preheat the oven to 175°C.
7. Bake for 10–15 minutes; the longer you bake, the crispier the cookies will be.
8. Let cool slightly before serving.
