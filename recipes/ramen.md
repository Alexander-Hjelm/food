## Ingredients

| Ingredient    | Amount  | Prep         | Optional |
| ------------- | ------- | ------------ | -------- |

| Broth cube    | 1       |              |          |

| Tahini        |         | 1 tbsp       |          |
| Miso paste    |         | 1.5 tbsp     |          |
| Soy sauce     |         | 1.5 tbsp     |          |
| Garlic        | 1 clove | minced       |          |
| Spring Onions |         |              |          |
| Sesame seeds  |         |              |          |
| Cabbage       |         |              | Optional |
| Bok Choi      |         |              | Optional |
| Corn          |         |              | Optional |
| Beans         |         |              | Optional |
| Bamboo shoots |         |              | Optional |
| Tofu          |         |              | Optional |
| Mushrooms     |         |              | Optional |
| Tree ears     |         |              | Optional |
| Bacon         |         |              | Optional |


## Instructions

This recipe is for 2 servings.

1. Bring the noodles to a boil together with water and broth cube.
2. In the serving bowl, mix tahini, miso paste, soy sauce and garlic with a little bit of the noodle water.
3. Pour noodles and soup into the serving bowls
4. Pepare and arrange remaining optional ingredients
