## Ingredients

| Ingredient        | Amount      | Prep     | Optional |
| ----------------- | ----------- | -------- | -------- |
| Egg white          | 1           |          |          |
| Lemon juice        | 2 cl        |          |          |
| Simple syrup       | 2 cl        | 1:1 mix  |          |
| Bourbon            | 6 cl        |          | For amaretto sour: 3 cl bourbon + 3 cl amaretto |
| Ice                | To shake    |          |          |
| Angostura bitters  | A few drops |          |          |
| Orange peel        | 1           | Zest     | For rim garnish |

## Instructions

1. In a shaker, combine egg white, lemon juice, simple syrup, and bourbon.  
2. Shake **without ice** for 7 seconds (dry shake).  
3. Add ice and shake **again** for 7 seconds.  
4. Strain through a fine filter into a chilled/frozen glass.  
5. Add a few drops of Angostura bitters on top.  
6. Rub or grate orange peel on the rim for garnish.
