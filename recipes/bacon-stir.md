# Bacon Stir

> Man prefers to believe what he prefers to be true.

## Tags

- swedish, main course, pasta

## Ingredients

| Ingredient       | Amount     | Prep      | Optional |
| ---------------- | ---------- | --------- | -------- |
| Bacon / Vegan bacon / Soy meat | 400 g      | diced     |          |
| Creme fraiche    | 2 dl       |          |          |
| Onion            | 2          | diced     |          |
| Oil              | 1 tbsp     |          |          |
| Macaroni         | 250 g      | cooked    |          |
| Salt             | to taste   |          | Optional |
| Pepper           | to taste   |          | Optional |

## Instructions

1. Fry bacon/meat in oil for a few minutes.
2. Add diced onions and cook until soft.
3. Stir in creme fraiche, season with salt and pepper, and serve with cooked macaroni.
