# How to: White Sauce

> ?

## Tags

- italian, sauce, guide

## Ingredients

| Ingredient | Amount      | Prep | Optional |
| ---------- | ----------- | ---- | -------- |
| Butter     | 50 g        |      |          |
| Wheat flour| 50 g        |      |          |
| Cream      | 500 ml      |      |          |
| Salt       | to taste    |      |          |
| Black pepper | to taste  |      | Optional |

## Instructions

1. Melt the butter gently in a saucepan over medium heat.
2. Whisk in the flour and cook for about 1 minute, stirring constantly, without letting it brown.
3. Gradually add the cream while whisking continuously to prevent lumps.
4. Continue heating and stirring until the sauce thickens to a smooth, creamy consistency.
5. Season with salt and optionally black pepper.
6. Use immediately or keep warm over low heat, stirring occasionally.
