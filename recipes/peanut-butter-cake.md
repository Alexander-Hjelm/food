# Peanut Butter Cake

> Rich, nutty, and slightly gooey—basically peanut butter in cake form.

## Tags

- dessert, baking, cake, sweet

## Ingredients

| Ingredient           | Amount       | Prep                  | Optional |
| -------------------- | ------------ | -------------------- | -------- |
| Wheat flour          | 3 cups       |                      |          |
| Sugar                | 1/2 cup      |                      |          |
| Salt                 | 2 tsp        |                      |          |
| Baking powder        | 1.5 tsp      |                      |          |
| Butter               | 1 cup        | melted                |          |
| Water                | 1 cup        |                      |          |
| Peanut butter        | 1.25 cup     |                      |          |
| Oil                  | 1 tsp        |                      | Optional |
| Eggs                 | 3            | beaten               |          |
| Milk                 | 2–3 tbsp     |                      | Optional |
| Vanilla extract      | 2 tsp        |                      |          |
| Nuts                 | 50 g         | chopped, for topping | Optional |

## Instructions

1. Preheat the oven to 175°C. Grease and flour a baking pan.
2. In a bowl, mix together the flour, sugar, salt, and baking powder.
3. In a separate pan, melt the butter with water over medium heat. Stir in peanut butter and a little oil until smooth.
4. In another bowl, whisk the eggs with milk and vanilla extract.
5. Combine all three mixtures, stirring until sticky and well incorporated.
6. Pour the batter into the prepared pan and sprinkle chopped nuts on top.
7. Bake for 20 minutes or until a toothpick inserted in the center comes out clean.
8. Allow the cake to cool before slicing and serving.
