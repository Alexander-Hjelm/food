# Kladdkaka (Gooey Chocolate Cake)

> He shit everywhere. THERE'S SHIT EVERYWHERE! DAMN IT!!! THERE'S SHIT ON THE WINDOWS! OH MY GOD! MY HOUSE IS FULL OF SHIT!! HE SHIT EVERYWHERE!! LOOK WHAT HE DID! HE SHIT ALL OVER THE WALL! GET UP HERE!

## Ingredients

| Ingredient        | Amount  | Prep   | Optional |
| ----------------- | ------- | ------ | -------- |
| Butter            | 100 g   | melted |          |
| Eggs              | 3       |        |          |
| Granulated sugar  | 1.5 dl    |        |          |
| Vanilla sugar     | 1 tbsp  |        |          |
| Cocoa powder      | 5 tbsp  |        |          |
| Wheat flour       | 1.5 dl  |        |          |
| Powdered Sugar    | 2 tbsp  |        |          |
| Whipped Cream     | 1.5 dl  |        |          |
| Vanilla extract   | 0.5 tsp |        |          |

## Instructions

1. Mix melted butter, eggs, sugar, vanilla sugar, cocoa, and flour into a smooth batter.
2. Pour into a greased glass baking dish.
3. Bake at 200 Â°C for about 40 minutes; cool slightly before serving.
4. Serve with powdered sugar and whipped cream with added vanilla extract.
