# General Tso Fried Cauliflower

> ?

## Tags

- chinese, vegan, main course, snack

## Ingredients

| Ingredient        | Amount          | Prep             | Optional |
| ----------------- | --------------- | ---------------- | -------- |
| Garlic            | 2 cloves        | minced           |          |
| Fresh ginger      | 1 Tbsp          | grated           |          |
| Vegetable broth   | 120ml           |                  |          |
| Soy sauce         | 60ml            |                  |          |
| Rice vinegar      | 60ml            |                  |          |
| Brown sugar       | 50g             |                  |          |
| Tomato paste      | 30g             |                  |          |
| Sesame oil        | 15ml            |                  |          |
| Cornstarch        | 15g             | dissolved in 30ml cold water |          |
| Cauliflower       | 1 head (approx 500g) | cut into florets |          |
| Flour             | Enough to coat  |                  |          |
| Eggs              | 2, whisked     |                  |          |
| Panko breadcrumbs | Enough to coat  |                  |          |

## Instructions

1. Preheat oven (temperature not specified, approx 200°C).  
2. Soak cauliflower florets in water briefly, then coat in flour, whisked eggs, and panko breadcrumbs.  
3. Bake in the oven for 15 minutes until crispy.  
4. In a pan, sauté garlic and ginger, add vegetable broth, soy sauce, rice vinegar, brown sugar, tomato paste, and sesame oil.  
5. Thicken sauce by adding cornstarch slurry and simmer until glossy.  
6. Toss baked cauliflower in sauce and serve immediately.
