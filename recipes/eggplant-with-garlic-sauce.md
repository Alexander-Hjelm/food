# Eggplant with Garlic Sauce

> ?

## Tags

- chinese, asian, main course, stir fry

## Ingredients

| Ingredient                | Amount      | Prep                          | Optional |
| ------------------------- | ----------- | ----------------------------- | -------- |
| Hot water                 | 120 ml      |                               |          |
| Sugar                     | 15 ml       |                               |          |
| Cornstarch                | 15 ml       |                               |          |
| Oyster sauce (veg ok)     | 22.5 ml     |                               |          |
| Rice vinegar              | 15 ml       |                               |          |
| Shaoxing wine             | 15 ml       |                               |          |
| Light soy sauce           | 15 ml       |                               |          |
| Sesame oil                | 5 ml        |                               |          |
| Dark soy sauce            | 2.5 ml      |                               |          |
| White pepper              | 0.5 ml      |                               |          |
| Garlic                    | 2 tbsp ~ 6 cloves | Minced (divided)       |          |
| Chinese/Japanese eggplants | ~450 g (2–3) | Cut into batons or wedges   |          |
| Neutral oil               | 67.5 ml     |                               |          |
| Ground pork/chicken       | 113 g       |                               | Optional |
| Ginger                    | 2 tsp       | Minced                        |          |
| Dried red chilies         | 5–10        | Deseeded, sliced              |          |
| Scallions                 | 2           | Chopped                       |          |

## Instructions

1. In a bowl, whisk together hot water, sugar, cornstarch, oyster sauce, rice vinegar, Shaoxing wine, light soy sauce, sesame oil, dark soy sauce, and white pepper. Set aside.
2. Cut the eggplants into long batons or wedges.  
3. Heat the neutral oil in a wok over medium–high heat. Add the eggplant and fry until softened and lightly golden. Remove and set aside.
4. In the same wok, add half of the minced garlic, the minced ginger, and the dried chilies. Stir-fry until fragrant.
5. If using ground pork or chicken, add it now and cook until browned.
6. Return the eggplant to the wok and pour in the prepared sauce. Stir to coat evenly.
7. Let the sauce thicken while tossing everything together.
8. Add the remaining garlic and the chopped scallions. Stir-fry for another minute.
9. Serve hot with rice or noodles.
