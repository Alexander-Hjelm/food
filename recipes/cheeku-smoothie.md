# Cheeku Smoothie

> The cheeky fruit gets cheekier when blended with banana and soy milk!

## Tags

- indian, smoothie, vegan, breakfast, drink, healthy

## Ingredients

| Ingredient   | Amount      | Prep      | Optional |
| ------------ | ----------- | -------- | -------- |
| Soy milk     | 250ml       | sweetened |          |
| Banana       | 1 medium    | peeled    |          |
| Cheeku fruit | 1-2         | peeled    |          |

## Instructions

1. Peel the banana and cheeku fruit.  
2. Add the banana, cheeku fruit, and soy milk into a blender.  
3. Blend until smooth and creamy.  
4. Pour into a glass and serve immediately.
