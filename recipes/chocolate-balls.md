# Chocolate balls

> Back in my day we used to call them [REDACTED] Balls!

## Tags

- dessert, swedish, baking, chocolate

## Ingredients

| Ingredient    | Amount  | Prep   | Optional |
| ------------- | ------- | ------ | -------- |
| Butter        | 100 g   | melted |          |
| Sugar         | 0.5 dl  |        |          |
| Cocoa powder  | 3 tbsp  |        |          |
| Oats          | 3 dl    |        |          |
| Strong coffee | 3 tbsp  |        |          |
| Coco shavings | 1.5 dl  |        | Optional |

## Instructions

1. Mix melted butter, sugar, cocoa, coffee and oats
2. Let the mix sit in the frige for 40 minutes +
3. Roll into balls of desired size and garnish with coco shavings
