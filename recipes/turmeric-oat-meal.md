# Turmeric Oatmeal

> Not the only yellow thing that makes mornings exciting, if you know what I mean.

## Tags

- breakfast, vegan, oatmeal

## Ingredients

| Ingredient       | Amount      | Prep       | Optional |
| ---------------- | ----------- | ---------- | -------- |
| Soy milk         | 240ml       |            |          |
| Rolled oats      | 90g         |            |          |
| Turmeric powder  | 0.5 tsp     |            |          |
| Cinnamon         | 0.5 tsp     |            |          |
| Cloves           | 0.125 tsp   |            |          |
| Black pepper     | 0.125 tsp   |            |          |
| Seeds            | To taste    |            | Optional |
| Berries          | To taste    |            | Optional |
| Apple            | 1           | sliced     | Optional |

## Instructions

1. Combine soymilk, rolled oats, turmeric, cinnamon, cloves, and black pepper in a pot.  
2. Cook over medium heat, stirring occasionally, until oats are soft and creamy.  
3. Serve topped with seeds, berries, and sliced apple.
