# Orange Cake

> When life gives you oranges… blend them into a cake and call it genius.

## Tags

- german, dessert, baking, cake, sweet

## Ingredients

| Ingredient      | Amount      | Prep       | Optional |
|-----------------|-------------|------------|----------|
| Eggs            | 3           |            |          |
| Sugar           | 100g        |            |          |
| Oranges         | 2 medium    | whole, blended |          |
| Flour           | 600g        |            |          |
| Butter          | 200g        | melted     |          |
| Butter (for pan)| as needed   |            |          |
| Breadcrumbs (for pan) | as needed |      |          |

## Instructions

1. Blend the eggs, sugar, and whole oranges until smooth.  
2. Add the flour and melted butter to the mixture and mix until fully combined.  
3. Grease a cake tin with butter and coat it lightly with breadcrumbs.  
4. Pour the batter into the prepared tin.  
5. Bake at 175°C for 40–50 minutes, or until a toothpick comes out clean.  
6. Let cool before slicing and serving.
