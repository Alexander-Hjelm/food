# Granola

> This recipe slaps harder than your dad.

## Tags

- breakfast, snack

## Ingredients

| Ingredient       | Amount       | Prep       | Optional |
| ---------------- | ----------- | ---------- | -------- |
| Oats             | 200g        |          |          |
| Seeds            | 50g         |          |          |
| Nuts             | 100g        | chopped   |          |
| Dried fruit      | 100g        | chopped   | Optional |
| Shredded coconut | 50g         |          | Optional |
| Cinnamon         | 1 tsp       |          | Optional |
| Oil              | 2 tbsp      |          |          |
| Honey            | 2 tbsp      |          | Optional |
| Salt             | 1/4 tsp     |          |          |

## Instructions

1. Preheat oven to 150Â°C. Mix oats, seeds, nuts, coconut, cinnamon, oil, honey, and salt in a bowl.  
2. Spread mixture on a baking sheet. Bake for 20 minutes, stirring after 10 minutes.  
3. Let cool completely and mix in dried fruit before serving.
