# Negroni Sbagliato

> ?

## Tags

- italian, cocktail, drink

## Ingredients

| Ingredient        | Amount | Prep                | Optional |
| ----------------- | ------ | ------------------- | -------- |
| Sweet vermouth    | 30 ml  |                     |          |
| Campari           | 30 ml  |                     |          |
| Gin               | 30 ml  |                     |          |
| Ice               | plenty |                     |          |
| Orange peel       | 1 strip| thinly cut          | Optional |

## Instructions

1. Chill a serving glass in the freezer.
2. Add ice to a shaker until full.
3. Pour vermouth, Campari, and gin into the shaker.
4. Stir until well chilled and taste with a spoon to check temperature.
5. Strain into the frozen glass over fresh ice.
6. Squeeze the orange peel over the drink, rub it around the rim, and drop it into the glass.
7. Serve immediately.
