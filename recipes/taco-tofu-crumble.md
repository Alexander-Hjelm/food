# Taco Tofu Crumble

> Tofu goes wild with taco night vibes.

## Tags

- vegan, mexican, main course, tofu

## Ingredients

| Ingredient    | Amount      | Prep     | Optional |
| ------------- | ----------- | -------- | -------- |
| Firm tofu     | 400g        | crumbled |          |
| Olive oil     | 2 tbsp      |          |          |
| Soy sauce     | 2 tbsp      |          |          |
| Taco spice mix| 1-2 tsp     |          |          |

## Instructions

1. Preheat oven to 150Â°C.  
2. Crumble tofu into a bowl and mix with olive oil, soy sauce, and taco spices.  
3. Spread tofu on a baking sheet and bake for 30 minutes until slightly crispy.
