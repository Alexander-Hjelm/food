# Pasta Sauce

> Doesn't fill you up at all, for fucks sake, but anyway.

## Tags

- italian, main course, sauce, vegetarian

## Ingredients

| Ingredient      | Amount | Prep       | Optional |
| --------------- | ------ | ---------- | -------- |
| Yellow onion    | 1      | finely chopped |          |
| Red bell pepper | 1      | sliced     |          |
| Red chili       | 1      | finely chopped |          |
| Tomatoes        | 2      | diced      |          |
| Crème fraîche   | 200 ml |            |          |
| Tomato paste    | 2 tbsp |            |          |
| Salt            | 1 tsp  |            |          |

## Instructions

1. Sauté onion in oil, then add pepper, tomatoes, and chili.
2. Cook a few minutes, then stir in remaining ingredients.
3. Simmer while stirring until creamy.
