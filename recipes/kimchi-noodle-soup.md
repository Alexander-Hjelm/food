# Kimchi Noodle Soup

> ?

## Tags

- korean, asian, soup, main course, vegan

## Ingredients

| Ingredient     | Amount      | Prep        | Optional |
| -------------  | ----------- | ----------- | -------- |
| Kimchi sauce   | 3 tbsp      |             |          |
| Miso paste     | 2 tbsp      |             |          |
| Tahini         | 1 tbsp      |             |          |
| Chili oil      | 1 tsp       |             |          |
| Salt           | to taste    |             |          |
| Pepper         | to taste    |             |          |
| Rice noodles   | 150g        | cooked      |          |
| Tofu           | 150g        | cubed       | Optional |
| Spring onions  | 2 stalks    | chopped     |          |

## Instructions

1. Mix kimchi sauce, miso paste, tahini, and chili oil in a pot with water; season with salt and pepper.  
2. Heat until warm and add tofu cubes.  
3. Add cooked rice noodles, stir, and top with chopped spring onions before serving.
