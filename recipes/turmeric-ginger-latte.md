## Ingredients

| Ingredient        | Amount        | Prep       | Optional |
| ----------------- | ------------- | ---------- | -------- |
| Milk               | 240ml        |            |          |
| Honey or maple syrup | 1 tbsp      |            |          |
| Ground turmeric     | 1 tsp       |            |          |
| Ground cinnamon     | 0.25 tsp    |            |          |
| Ground cardamom     | 0.125 tsp   |            |          |
| Ground nutmeg       | 0.125 tsp   |            |          |
| Ground ginger       | 0.25 tsp    |            | Optional |

## Instructions

1. Heat milk in a small saucepan over medium heat until warm but not boiling.  
2. Whisk in honey (or maple syrup), turmeric, cinnamon, cardamom, nutmeg, and ginger if using.  
3. Pour into a mug and enjoy warm.
