# Pasta Carbonara

> The ultimate Italian “don’t overcook the eggs” challenge.

## Tags

- italian, main course, pasta

## Ingredients

| Ingredient    | Amount      | Prep     | Optional |
| ------------- | ----------- | -------- | -------- |
| Pasta         | 100g per person |          |          |
| Bacon         | 50-100g per person | chopped |          |
| Garlic        | 1 clove per person | minced |          |
| Eggs          | 1 per person |          |          |
| Parmesan cheese | 30g per person | grated |          |
| Black pepper  | To taste   | ground  |          |
| Salt          | To taste   |          |          |
| Parsley       | For garnish | chopped | Optional |

## Instructions

1. Fry bacon until crispy, add garlic and black pepper near the end.  
2. Boil pasta and reserve some pasta water.  
3. Remove bacon from heat, toss in pasta and a splash of pasta water until pasta reaches ~70°C.  
4. Mix eggs and grated cheese in a bowl.  
5. Remove pan from heat, quickly stir in egg-cheese mixture until creamy.  
6. Top with salt, parsley, and extra cheese before serving.
