## Ingredients

| Ingredient  | Amount  | Prep     | Optional |
| ----------- | --------| -------- | -------- |
| Egg yolks   | 3       |          |          |
| Milk        | 400 ml  |          |          |
| Corn starch | 20g     |          |          |
| Sugar       | 60g     |          |          |
| Cinnamon    | 1 stick |          |          |
| Lemon zest  |         |          |          |
| Lemon juice |         |          |          |

## Instructions

1. Whisk egg yolks, corn starch and 100 ml milk and set aside
2. Mix remaining milk with sugar, cinnamon stick, lemon zest and lemon juice and bring to a boil.
3. Take the pot off the heat and slowly mix in the egg yolk mixture while stirring.
4. Bring to a boil again.
5. Pour into bowls and serve warm.
