# Minestrone Soup

> ?

## Tags

- italian, soup, main course, vegetarian

## Ingredients

| Ingredient       | Amount      | Prep       | Optional |
| ---------------- | ----------- | ---------- | -------- |
| Carrots          | 200g        | diced      |          |
| Celery           | 150g        | diced      |          |
| Garlic           | 2 cloves    | minced     |          |
| Onion            | 1           | minced     |          |
| Crushed tomatoes | 400g        |            |          |
| Salt             | 1 tsp       |            |          |
| Pepper           | 0.5 tsp     |            |          |
| Vegetable broth  | 1 cube      | dissolved  |          |
| Cheese           | 50g         | grated     | Optional |
| Bread            | 1 slice     |            | Optional |

## Instructions

1. Saute diced carrots and celery in a little oil until slightly softened.  
2. Blend garlic and onion, then saute until fragrant.  
3. Add everything to a pot with crushed tomatoes and dissolved broth cube. Season with salt and pepper.  
4. Bring to a boil, then blend to desired consistency.  
5. Serve hot with grated cheese and bread.
