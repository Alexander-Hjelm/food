# Veggie balls

> Alternatively, do it like a proper lumberjack and fry in butter flavuored vape juice. Yummy!

## Tags

- vegetarian, main course, swedish

## Ingredients

| Ingredient    | Amount      | Prep       | Optional |
| ------------- | ----------- | --------- | -------- |
| Chickpeas     | 1 can       | drained   |          |
| Potatoes      | 4           | boiled    |          |
| Onion         | 0.5         | chopped   |          |
| Mushrooms     | 6â€“7         | chopped   |          |
| Breadcrumbs   | as needed   |           |          |
| Butter        | 2 tbsp      |           | Optional |

## Instructions

1. Boil potatoes until tender, then mash them together with chickpeas, onion, and mushrooms in a food processor.
2. Form the mixture into balls, then roll them in breadcrumbs.
3. Fry the balls in butter over medium heat until golden and cooked through.
