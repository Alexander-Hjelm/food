# Banana Pancakes (with Cocoa)

> ?

## Tags

- breakfast, dessert, pancakes, vegetarian

## Ingredients

| Ingredient | Amount | Prep | Optional |
| ---------- | ------ | ---- | -------- |
| Wheat flour | 250 g | | |
| Baking powder | 2 tsp | | |
| Salt | 0.5 tsp | | |
| Cocoa powder | 50 g | | |
| Bananas | 230 g | mashed | |
| Milk | 300 g | | |
| Eggs | 2 | | |
| Butter | 100 g | melted | |

## Instructions

1. In a bowl, mix the dry ingredients: flour, baking powder, salt, and cocoa powder.
2. In another bowl, mix the wet ingredients: mashed bananas, milk, eggs, and melted butter.
3. Combine the wet and dry ingredients and stir until just mixed.
4. Heat a pan over medium heat and lightly grease it.
5. Pour batter into the pan and cook until bubbles form, then flip and cook until done.
6. Serve warm with your favorite toppings.
