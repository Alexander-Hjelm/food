# Cabbage Soup

> The soup that makes you feel like a cozy Swedish winter is happening, even if it’s not.

## Tags

- swedish, soup, main course, vegetarian

## Ingredients

| Ingredient        | Amount      | Prep                  | Optional |
| ----------------- | ----------- | -------------------- | -------- |
| White cabbage      | 1 head     | Chopped               |          |
| Carrots            | 2-3        | Chopped               |          |
| Potatoes           | 2-3        | Chopped               |          |
| Onion              | 1 large    | Chopped               |          |
| Sirap / syrup      | 1-2 tbsp   |                       | Optional |
| Butter or cooking fat | 2 tbsp  |                       |          |
| Water              | 1.5 L      |                       |          |
| Vegetable bouillon cubes | 4     |                       |          |
| Black pepper       | To taste   | Grated                |          |
| Bay leaves         | 2-3        |                       |          |
| Allspice berries   | 4-5        |                       | Optional |
| Spring onions      | 2          | Sliced                |          |
| Bread & cheese     | To serve   |                       | Optional |

## Instructions

1. Chop and sauté the cabbage in a little syrup and butter or cooking fat. Add chopped onion and cook lightly towards the end.  
2. Chop carrots and potatoes. Prepare the bouillon by dissolving the cubes in 1.5 L water with some grated black pepper.  
3. Add all vegetables to the bouillon along with bay leaves and allspice berries. Simmer until vegetables are tender.  
4. Add sliced spring onions just before serving.  
5. Serve hot with a cheese sandwich on the side.
