# Pickled Herring

> Seriously, your whole kitchen will smell like ass for weeks to come. Don't say I didn't warn you.

## Tags

- swedish, scandinavian, snack, christmas

## Ingredients

| Ingredient                   | Amount    | Prep   | Optional |
| ---------------------------- | --------- | ------ | -------- |
| ABBA 5 minute herring (430g) | 1 package |        |          |
| Red onion                    | 1/4       | sliced |          |
| Carrot                       | 1/2       | sliced |          |
| Spring onion                 | 1         | sliced |          |
| White peppercorn             | 1 tsp     |        |          |
| Cloved                       | 1 tsp     |        |          |
| Bay leaves                   | 2         |        |          |
| Mustard seeds                | 1/2 tsp   |        |          |
| Parsley stems                |           |        | Optional |
| Dill                         |           |        | Optional |

## Instructions

1. Separate the herring bits from the pickle brine. Save the brine for later.
2. Bring the brine to a boil and set aside.
3. In a large airtight jar, layer all flavourings together with the herring.
4. Store in the fridge. Ready to eat in 2 days.
