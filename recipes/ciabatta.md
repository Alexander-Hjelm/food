# Ciabatta

> ?

## Tags

- bread, italian, baking, main course, snack

## Ingredients

| Ingredient     | Amount      | Prep      | Optional |
| -------------- | ----------- | -------- | -------- |
| Flour          | 670g        |          | Can mix different types |
| Sugar          | 30g         |          |          |
| Olive oil      | 45g         |          |          |
| Salt           | 20g         |          |          |
| Oats           | To taste    |          | Optional |
| Yeast          | 20g         | Active in water |          |
| Water          | 450g        |          |          |

## Instructions

1. Activate 20g of yeast in 450g of water until foamy.  
2. Mix the activated yeast with flour, sugar, oil, and salt to form a dough.  
3. Knead the dough well and let it rise for 30 minutes.  
4. Shape the dough onto a baking tray, make slashes on top, and sprinkle with flour and optional oats.  
5. Bake in a preheated oven for 20 minutes until golden brown.
