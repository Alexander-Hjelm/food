# Daiquiri

> ?

## Tags

- cocktail, drink, caribbean

## Ingredients

| Ingredient     | Amount | Prep        | Optional |
| -------------- | ------ | ----------- | -------- |
| Lime juice     | 25 ml  | freshly squeezed |        |
| Simple syrup   | 20 ml  | 1:1 sugar/water |        |
| White rum      | 50 ml  |             |          |
| Lime peel      | 1 strip| garnish     | Optional |

## Instructions

1. Add lime juice, simple syrup, and rum to a shaker.
2. Fill the shaker with ice.
3. Shake hard for about 7 seconds.
4. Strain into a frozen cocktail glass.
5. Garnish with lime peel and serve cold.
