# Honey-Glazed Chicken

> ?

## Tags

- asian, main course

## Ingredients

| Ingredient       | Amount       | Prep          | Optional |
| ---------------- | ----------- | ------------- | -------- |
| Chicken breasts  | 4           | thawed, salted, drained |          |
| Honey            | 120 ml      |               |          |
| Light soy sauce  | 60 ml       |               |          |
| Chili flakes     | 1 tsp       |               | Optional |
| Salt             | to taste    |               |          |
| Black pepper     | to taste    |               | Optional |
| Oil              | 3 tbsp      |               |          |
| Rice             | 2 cups      |               |          |
| Water            | as needed   | up to surface + 1 hand thickness | |

## Instructions

1. Preheat the oven or pan to medium heat.
2. Season the thawed chicken breasts with salt and pepper.
3. In a small bowl, mix honey, light soy sauce, and chili flakes.
4. Heat oil in a pan and sear the chicken breasts on both sides until lightly browned.
5. Brush the honey-soy mixture over the chicken and continue cooking until fully glazed and cooked through.
6. Meanwhile, rinse the rice and add it to a rice cooker.
7. Pour water into the rice cooker up to the surface of the rice plus about one handâ€™s thickness.
8. Cook the rice according to the rice cooker instructions.
9. Serve the honey-glazed chicken over the cooked rice.
