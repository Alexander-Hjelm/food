# French 75

> ?

## Tags

- cocktail, alcoholic, drink, summer, winter

## Ingredients

| Ingredient      | Amount      | Prep     | Optional |
| --------------- | ----------- | -------- | -------- |
| Lemon juice     | 2 cl        |          |          |
| Simple syrup    | 2.5 cl      | 1:1 mix  |          |
| Gin or Cognac   | 5 cl        |          | Gin for summer, Cognac for winter |
| Ice             | To fill     |          |          |
| Champagne       | 6 cl        | chilled  |          |
| Lemon zest      | 1 small strip |         | For garnish |

## Instructions

1. Fill a shaker to the brim with ice.  
2. Add lemon juice, simple syrup, and gin or cognac.  
3. Shake hard for 7 seconds.  
4. Strain the mixture into a chilled champagne glass.  
5. Top with 6 cl of chilled champagne or sparkling wine.  
6. Press a small piece of lemon zest over the drink and drop it in as garnish.
