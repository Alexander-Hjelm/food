# Cake

> ?

## Tags

- dessert, swedish, cake

## Ingredients

| Ingredient        | Amount      | Prep               | Optional |
| ----------------- | ----------- | ------------------ | -------- |
| Sponge cake base  | 1           | store-bought       |          |
| Custard (vanilla) | 300 ml      | prepared           |          |
| Jam               | 200 ml      |                    |          |
| Whipping cream    | 500 ml      | whipped            |          |
| Marzipan lid      | 1           |                    |          |

## Instructions

1. Place the sponge cake base on a serving plate.
2. Mix custard, jam, and part of the whipped cream.
3. Spread the mixture evenly over the cake base.
4. Cover with an additional layer of whipped cream.
5. Place the marzipan lid carefully on top.
6. Chill before serving.
